🎯 핵심 결론 (Bottom Line)
================================================================================

▶ ICP (타겟 고객)
  • 독립 카페/소매점 점장 (1-5개 매장)
  • 월 매출: $50K-500K
  • Pain Point: 관리 어려움 + 재방문 추적 불가 + 높은 비용

▶ Solution
  • 블록체인 기반 토큰 로열티 플랫폼
  • MVP 5개 기능 (POS 연동, 자동 발급, no-wallet 앱, QR, Admin)

▶ Pricing
  • Starter: $499/월 (1-2 매장)
  • Growth: $799/월 (2-5 매장)
  • Scale: $1,199/월 (5+ 매장)
  • 기존 솔루션의 75% 저렴

▶ Unit Economics
  • ACV: $9.3K/년
  • CAC: $450
  • LTV: $10,900
  • LTV:CAC = 24:1 ✅ (탁월)
  • Payback Period: 1개월

▶ GTM
  • Model: Founder-Led Sales (B2B)
  • Sales Cycle: 2-4주
  • Month 6 Target: 15명 고객, $11.6K MRR
  • Year 1 Target: 50명 고객, $100K ARR

▶ Tech Stack
  • Backend: Node.js + Express + PostgreSQL
  • Frontend: React + Tailwind + Openfort SDK
  • Blockchain: Polygon (ERC-20)
  • Infrastructure: Railway + Vercel + Supabase

▶ Timeline
  • MVP 개발: 4-6주 (1인 개발자)
  • First 10 Customers: 3개월
  • $5K MRR: 6개월

💡 핵심 인사이트 (Key Insights)
================================================================================

1. 블록체인은 "기술"이 아니라 "신뢰 도구"
   → 고객은 기술을 모르고, 투명성만 느낌

2. No-Wallet UX는 필수 (고객이 암호를 모름)
   → Openfort SDK가 핵심 (embedded wallet)

3. Founder Sales는 빠르다 (2-4주 sales cycle)
   → 매장주는 B2B 구매자, Founder와 직접 거래

4. Unit Economics가 모든 것을 결정한다
   → CAC $450, Payback 1개월 = 무조건 스케일함

5. 기술 스택 선택이 운명을 결정한다
   → Node.js + React + Polygon = 1인 가능
