## 활용 방법

### 1. 주간 루틴
```
월: a16z Newsletter, YC Hacker News 체크
화: Crypto (a16z crypto, Paradigm, Messari)
수: AI (OpenAI, Anthropic blog)
목: VC blogs (Sequoia, Lightspeed)
금: Newsletter 정리 (Lenny, Stratechery)
```

### 2. 아이디어 검증 시
1. **문제 발굴**: YC RFS, a16z Big Ideas에서 트렌드 확인
2. **시장 검증**: CB Insights, Crunchbase에서 펀딩 동향
3. **기술 검증**: 각 분야 전문 소스에서 feasibility
4. **경쟁 분석**: Crunchbase, PitchBook에서 유사 스타트업

### 3. ERC-8004 관련 특별 트래킹
- a16z Crypto → KYA, Agent Wallets 관련 글
- EIPs → 에이전트 관련 새 표준
- Paradigm → DeFi + AI 교차점
- Messari → 에이전트 프로토콜 리서치
