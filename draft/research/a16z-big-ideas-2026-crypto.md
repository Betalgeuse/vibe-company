## 17 Big Ideas Summary

### 1. Privacy as the Most Important Moat (Ali Yahya)
- **핵심**: 프라이버시가 블록체인의 핵심 차별화 요소
- **인사이트**: 퍼블릭 체인 간 이동은 쉽지만, 프라이버시 체인은 lock-in 효과 발생
- **기회**: Winner-take-most 구조 → 프라이버시 체인이 크립토 대부분 차지할 수 있음

### 2. Prediction Markets Go Bigger, Broader, Smarter (Andrew Hall)
- **핵심**: 예측 시장이 더 많은 계약, AI 에이전트 트레이딩으로 확장
- **인사이트**: 분쟁 해결을 위한 분산화된 거버넌스 + LLM 오라클 필요
- **기회**: AI 에이전트가 예측 시장에서 트레이딩, 새로운 신호 발굴

### 3. Crypto-Native RWA Tokenization (Guy Wuollet)
- **핵심**: Perpification vs Tokenization - 페퍼츄얼이 더 crypto-native
- **인사이트**: 이머징 마켓 주식 perpify가 흥미로운 자산군
- **기회**: 온체인 origination (오프체인 토큰화 X)

### 4. Trading is Not the Final Destination (Arianna Simpson)
- **핵심**: 모든 크립토 회사가 트레이딩으로 피봇하는 건 위험
- **인사이트**: 즉각적 PMF 추구의 함정 (마시멜로 테스트)
- **교훈**: "Product" 부분에 집중하는 창업자가 더 큰 승자

### 5. ⭐ KYC → KYA (Know Your Agent) (Sean Neville - Circle 공동창업자)
- **핵심**: 에이전트 경제의 병목이 intelligence → identity로 이동
- **인사이트**: 금융서비스에서 non-human identity가 인간 직원의 96:1 비율
- **기회**: 에이전트에게 cryptographically signed credentials 필요
- **연관성**: ERC-8004 에이전트 신용평가와 직접 연결

### 6. Better Stablecoin On/Off Ramps (Jeremy Zhang)
- **핵심**: 스테이블코인 연간 거래량 $46T (PayPal 20배, Visa 3배)
- **인사이트**: 디지털 달러와 기존 금융 레일 연결이 미해결
- **기회**: QR코드, 실시간 결제, 글로벌 월렛 레이어

### 7. Stablecoins Unlock Bank Ledger Upgrade (Sam Broner)
- **핵심**: 은행 코어 레저가 60-70년대 COBOL 기반
- **인사이트**: 스테이블코인으로 레거시 시스템 재작성 없이 혁신 가능
- **기회**: 토큰화된 예금, 국채, 온체인 채권

### 8. Decentralized Quantum-Resistant Messaging (Shane Mac - XMTP)
- **핵심**: 양자 암호화만으로는 부족, 탈중앙화 필요
- **인사이트**: 중앙 서버 = 정부 차단/백도어 타겟
- **기회**: 메시지 소유권 (키로 소유)

### 9. 'Spec is Law' - DeFi Security Evolution (Daejun Park)
- **핵심**: "Code is law" → "Spec is law"
- **인사이트**: AI 기반 formal verification 도구로 invariant 증명
- **기회**: 런타임 가드레일로 해킹 자동 차단

### 10. SNARKs Beyond Blockchains (Justin Thaler)
- **핵심**: 2026년 zkVM 프루버 오버헤드 10,000X로 감소
- **인사이트**: GPU 1개로 실시간 CPU 실행 증명 가능
- **기회**: Verifiable cloud computing

### 11. AI for Substantive Research (Scott Duke Kominers)
- **핵심**: AI가 박사과정 학생처럼 추상적 지시 수행 가능
- **인사이트**: Agent-wrapping-agent 워크플로우 필요
- **기회**: 모델 간 interoperability + 기여도 보상 (크립토로 해결)

### 12. The Invisible Tax on Open Web (Elizabeth Harkavy)
- **핵심**: AI 에이전트가 광고 기반 웹에서 데이터 추출, 수익 우회
- **인사이트**: 정적 라이선싱 → 실시간 usage-based 보상으로 전환 필요
- **기회**: 블록체인 기반 nanopayment + attribution 시스템

### 13. The Rise of Staked Media (Robert Hackett)
- **핵심**: "Skin in the game"을 증명하는 미디어
- **인사이트**: AI가 무한 콘텐츠 생성 가능 → 신뢰 기반 필요
- **기회**: 토큰화된 자산, 예측 시장으로 credibility 증명

### 14. DeFi Goes Niche (Jason Rosenthal)
- **핵심**: 범용 DeFi → 틈새 시장 특화
- **인사이트**: 전통 금융이 아직 잘 서비스하지 못하는 영역
- **기회**: 특정 산업/지역/자산군 특화 DeFi

### 15. Agents Will Need Wallets (Sam Kim)
- **핵심**: AI 에이전트에게 자체 지갑 필요
- **인사이트**: 에이전트가 자율적으로 거래하려면 자금 관리 필요
- **기회**: 에이전트 전용 월렛 인프라

### 16. Crypto Market Structure Legislation (Miles Jennings)
- **핵심**: 규제 명확화가 네트워크를 네트워크답게 운영하게 함
- **인사이트**: "Enforcement roulette" → 구조화된 경로
- **기회**: 토큰 출시, 탈중앙화를 위한 명확한 규제

### 17. We'll Build More Onchain (Carra Wu)
- **핵심**: 더 많은 것들이 온체인으로 구축될 것
- **인사이트**: 인프라 성숙 → 애플리케이션 폭발
- **기회**: 온체인 네이티브 앱 개발
