## 1. The "Micro" Opportunity

### 🎯 타겟 고객 (ICP)
| 세그먼트 | 특징 | 규모 |
|----------|------|------|
| **1차 타겟** | 한국 IT 프리랜서 (개발자, 디자이너) | ~15만명 |
| **2차 타겟** | 비즈니스 컨설턴트, 마케팅 프리랜서 | ~10만명 |
| **3차 타겟** | 1-3인 소규모 에이전시 | ~5만개사 |
| **글로벌 확장** | 영어권 프리랜서 | 7천만명+ |

### 😤 Pain Point (문제점)

1. **행정 업무 과다**: 프리랜서의 40%+ 시간이 비생산적 행정 업무에 소비
2. **비전문적 이미지**: 카카오톡/이메일로 산발적 커뮤니케이션 → 신뢰도 하락
3. **제안서 작성 고통**: 매번 처음부터 제안서 작성 → 3-5시간 소요
4. **결제 추적 어려움**: 엑셀로 관리하다 미수금 놓침
5. **도구 파편화**: Notion + 구글드라이브 + 카카오톡 + 스프레드시트 혼용

### 💰 TAM/SAM/SOM
| 시장 | 규모 | 산출 근거 |
|------|------|-----------|
| **TAM** | $5.2B | 글로벌 프리랜서 관리 소프트웨어 시장 (2026) |
| **SAM** | $500M | 아시아 태평양 프리랜서 SaaS 시장 |
| **SOM** | $2M | 한국 프리랜서 1% 침투 × $15/월 × 12개월 |

## 2. Product Spec (MVP)

### 🔧 핵심 기능 (MVP)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **AI 제안서 생성** | 프로젝트 정보 입력 → AI가 맞춤형 제안서 자동 생성 | P0 |
| **클라이언트 포털** | 고객별 전용 페이지 (프로젝트 상태, 파일, 메시지) | P0 |
| **자동 인보이스** | 마일스톤 완료 시 자동 인보이스 발행 | P0 |
| **전자서명** | 계약서 템플릿 + 간편 전자서명 | P1 |
| **결제 연동** | 토스페이먼츠, Stripe 연동 | P1 |

### ⚡ 차별화 포인트

1. **AI 네이티브**: 제안서, 계약서, 이메일 모두 AI가 초안 생성 (경쟁사 대비 시간 50% 절감)
2. **한국 최적화**: 원화 인보이스, 사업자등록증 연동, 세금계산서 발행 지원
3. **초심플 UX**: Notion 수준의 직관적 인터페이스 (경쟁사 복잡함 해결)
4. **화이트라벨**: 내 브랜드 도메인으로 포털 운영 가능

### 🛠 Tech Stack (권장)

| 레이어 | 기술 | 이유 |
|--------|------|------|
| Frontend | Next.js + Tailwind | 빠른 개발, SEO 친화적 |
| Backend | Supabase (PostgreSQL + Auth) | 서버리스, 빠른 프로토타이핑 |
| AI | OpenAI GPT-4 API | 최고 품질 텍스트 생성 |
| 결제 | Stripe + 토스페이먼츠 | 글로벌 + 한국 커버 |
| 호스팅 | Vercel | 무료 시작, 스케일링 용이 |
| 이메일 | Resend | 트랜잭셔널 이메일 |

## 2.5 Competition Check (경쟁 분석)

### 🏆 직접 경쟁사

| 경쟁사 | 가격 | 강점 | 약점 |
|--------|------|------|------|
| **Moxie** | $29/월 | 올인원, 브랜딩 | 한국 미지원, 복잡 |
| **HoneyBook** | $19/월 | 이쁜 UI | 한국 결제 미지원 |
| **SuperOkay** | $9/월 | 저렴 | 기능 제한 |
| **Bonsai** | $25/월 | 계약 특화 | AI 없음 |

### 🇰🇷 한국 시장 상황
- **직접 경쟁사 부재**: 해외 서비스들이 한국 결제/세금 미지원
- **간접 경쟁**: 채널톡 (고객지원), 세금계산서 서비스 (정석), Notion (올라운더)
- **기회**: 한국 프리랜서를 위한 "맞춤형" 솔루션 부재

### 🛡 Why They Won't Crush You

1. **로컬라이제이션 해자**: 원화, 세금계산서, 사업자등록 연동은 글로벌 서비스가 쉽게 따라오지 못함
2. **AI 차별화**: 기존 경쟁사들은 템플릿 기반, 우리는 AI 생성 기반
3. **가격 경쟁력**: 한국 구매력 맞춤 가격 ($15 vs $29)
4. **니치 집중**: "한국 IT 프리랜서"라는 명확한 ICP
