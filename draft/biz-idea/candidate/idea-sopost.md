## 2. Product Spec (MVP)

### 🔧 Core Function
```
[입력] 뉴스레터/블로그 글 (URL 또는 텍스트)
           ↓
      AI 분석 & 변환
           ↓
[출력] 
  ├─ LinkedIn 포스트 (3가지 톤 옵션)
  ├─ X(트위터) 스레드 (5-7개 트윗)
  ├─ 인스타그램 캐러셀 텍스트 (10장 슬라이드)
  └─ 스레드(Threads) 포스트
```

### ⚡ MVP Features (4주 개발)
1. **원클릭 변환**: URL 입력 → 4개 플랫폼 콘텐츠 자동 생성
2. **한국어 톤 최적화**: 존댓말/반말, 이모지 스타일 선택
3. **플랫폼별 해시태그**: AI 추천 해시태그 자동 삽입
4. **브랜드 톤 저장**: 내 스타일 학습 및 일관성 유지
5. **예약 발행 연동**: Buffer/Later API 연동 (직접 발행 X)

### 🛠 Tech Stack
| 구분 | 기술 |
|------|------|
| Frontend | Next.js 14 + Tailwind CSS |
| Backend | Supabase (Auth, DB, Storage) |
| AI | OpenAI GPT-4o API |
| Payments | Stripe (한국은 Toss Payments 추가) |
| Hosting | Vercel |
| 예상 월 비용 | $50-100 (0-1000 사용자) |

### 🎨 Differentiation
| 경쟁사 | 한계 | SoPost 차별점 |
|--------|------|--------------|
| Repurpose.io | 비디오 중심, 텍스트 약함 | 텍스트→텍스트 특화 |
| ContentRepurpose.pro | 한국어 미지원 | 한국어 톤 완벽 지원 |
| Munch | 비디오→비디오 전용 | 블로그/뉴스레터 전문 |
| ChatGPT 직접 사용 | 발행 불가, 매번 프롬프트 입력 | 원클릭 + 스케줄링 연동 |
