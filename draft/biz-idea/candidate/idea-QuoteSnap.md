## 1. The "Micro" Opportunity

### 📌 한줄 설명
현장 사진 업로드만으로 AI가 견적서를 자동 생성하고, 스마트 팔로업으로 계약 전환율을 높여주는 서비스업 전용 SaaS

### 🎯 Target Market
- **ICP (Ideal Customer Profile)**: 
  - 인테리어/리모델링 업체
  - 이사/청소 업체
  - 설비/수리 업체 (전기, 배관 등)
  - 조경/외부 공사 업체
- **지역**: 한국 우선 (로컬 서비스업)
- **규모**: 1-10인 소규모 서비스 업체

### 💰 TAM Estimate
- 한국 건설/인테리어 관련 사업자: 약 50만개
- 소규모 서비스업 (1-10인): 약 30만개
- 견적 관리 니즈 있는 비율: 약 40% = 12만개
- 월 ₩29,000-59,000 가격대 → **월 MRR 35억원+ 가능**

### 😫 핵심 Pain Point
> "현장 다녀와서 밤늦게 엑셀로 견적서 만드는데, 너무 시간 낭비예요. 그리고 보내고 나서 팔로업 타이밍도 애매해요"

네이버 카페/블로그에서 반복 등장:
- 현장 방문 후 견적서 작성에 1-2시간 소요
- 카톡으로 보내면 읽었는지 확인 안됨
- 팔로업 시기를 놓쳐서 계약 기회 상실

## 2. Product Spec (MVP)

### 🔧 Core Functions
1. **사진 기반 AI 견적**: 
   - 현장 사진 업로드 → AI가 작업 항목 인식
   - 템플릿 기반 견적서 자동 생성
   - 단가표 학습으로 점점 정확해짐
2. **스마트 팔로업**: 
   - 카톡/문자 견적서 발송
   - 열람 추적 (링크 클릭 시 알림)
   - 3일/7일 후 자동 리마인더
3. **계약 전환 대시보드**:
   - 견적 → 계약 전환율 추적
   - 평균 응답 시간 분석
   - 성사율 높은 고객 유형 인사이트

### 🆚 Differentiation (vs 경쟁)
| 기존 솔루션 | 문제점 | QuoteSnap 차별화 |
|-------------|--------|------------------|
| 엑셀/한글 | 수동, 시간 소요 | AI 사진 분석 → 5분 내 완성 |
| 네이버 폼 | 심플하지만 추적 안됨 | 열람 추적 + 팔로업 자동화 |
| 해외 도구 (Jobber) | 한국 결제 미지원 | 카톡/문자 연동, 한국 특화 |

### 🛠️ Tech Stack
- **Frontend**: Next.js + Tailwind CSS
- **Backend**: Supabase
- **AI**: GPT-4 Vision (사진 → 작업 항목 추출)
- **알림**: 카카오 알림톡 + 문자 API (Toast)
- **결제**: 토스페이먼츠 (한국)
- **인프라**: Vercel

## 4. Why This Works for YOU (상혁님)?

### ✅ Low Maintenance
- 핵심 로직: 사진 → AI → 견적 템플릿
- 업종별 템플릿 한 번 세팅하면 유지보수 최소
- 카카오 알림톡은 API만 연동하면 자동

### ✅ Scalability
- GPT-4 Vision 비용: 이미지당 $0.01 이하
- 고객 증가 → 인프라 비용만 선형 증가
- 업종 확장: 템플릿 추가만 하면 됨

### ✅ Viral Elements
- "QuoteSnap으로 만든 견적서" 브랜딩 (무료 플랜)
- 고객에게 전문적인 견적서 → 신뢰도 상승 → 입소문
- 협회/모임 단위 확산 가능
