## 1. The "Micro" Opportunity
> 한국 스타트업/프리랜서가 Stripe로 해외 결제 받으면, 매달 수동으로 더존/세무사랑에 입력해야 함. 세무사 비용 + 시간 낭비.

- **Target:** Stripe 사용하는 한국 스타트업, 프리랜서, 1인 기업
- **Pain Point:** 해외 결제 내역 → 한국 회계 프로그램 수동 입력 지옥 (월 2-4시간)
- **TAM:** 한국 Stripe 가맹점 추정 5만+, 글로벌 확장 시 수백만

### 선택: Option A
- **Core Function:** Stripe 거래 내역 → 더존/세무사랑 호환 CSV/엑셀 자동 변환 + 다운로드
- **Differentiation:** 
  - 글로벌: QuickBooks 연동 많음
  - 한국: 더존/세무사랑 연동 **전무**
- **Tech Stack:** 
  - Next.js + Supabase (DB/Auth)
  - Stripe API (webhook)
  - Vercel (호스팅)
  - 토스페이먼츠 (국내 결제)
