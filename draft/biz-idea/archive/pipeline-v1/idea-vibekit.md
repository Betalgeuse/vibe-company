## 1. The "Micro" Opportunity
> 스마트스토어 셀러가 수백 개 리뷰를 일일이 읽으며 인사이트를 뽑는 데 3시간 이상 소요.

- **Target:** 네이버 스마트스토어 셀러 (50만명+)
- **Pain Point:** "리뷰 분석할 시간이 없어서 고객 불만 파악 못함"
- **Value Proposition:** "리뷰 분석 3시간 → 3분으로 단축"
- **TAM:** 스마트스토어 셀러 50만명, 유료 전환 0.1% = 500명

---

## 2. Product Spec (MVP)

- **Core Function:** 리뷰 CSV 업로드 → AI가 감정분석 + 키워드 추출 + 개선 액션 추천
- **Differentiation:** 
  - 한국어 감정분석 최적화
  - 단순 분석이 아닌 **"이렇게 고치세요"** 액션 가이드
  - 네이버 스마트스토어 특화 (상품명/상세페이지 개선 추천)
- **Tech Stack:** 
  - Next.js + Supabase + Vercel
  - Claude/GPT API (감정분석 + 요약)
  - Stripe/토스페이먼츠 (결제)

## 4. Why This Works for YOU?

- **Low Maintenance:** 
  - Serverless (Vercel + Supabase)
  - 크롤링 없음 → 차단 리스크 없음
  - API 호출 기반, 자체 서버 불필요
- **Scalability:** 
  - 리뷰 100개 분석당 비용 약 ₩100-200 (GPT API)
  - 마진 90% 이상 확보 가능
- **Viral:** 
  - 리포트에 워터마크 삽입
  - 셀러 커뮤니티에서 공유 유도
