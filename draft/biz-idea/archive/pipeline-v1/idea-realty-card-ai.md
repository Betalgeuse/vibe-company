## 1. The "Micro" Opportunity
> 공인중개사가 매물 홍보 콘텐츠 만들 시간/스킬이 없다. 사진만 보내면 카드뉴스가 나오길 원한다.

- **Target:** 한국 공인중개사, 부동산 중개 보조원
- **Pain Point:** "매물 사진은 있는데, 인스타/블로그에 올릴 예쁜 카드뉴스 만들 시간이 없음"
- **TAM:** 공인중개사 13만명+ (한국공인중개사협회 기준)

## 2. Product Spec (MVP)

- **Core Function:** 카카오톡 챗봇으로 매물 사진 전송 → AI가 카드뉴스 이미지 자동 생성
- **Differentiation:** 
  - 카카오톡 UI (학습 비용 0, IT 친숙도 낮은 타겟 최적화)
  - 부동산 특화 카피 (평수, 층수, 방향, 특징 자동 인식)
  - 네이버 부동산 감성 디자인 템플릿
- **Tech Stack:** 
  - 카카오톡 채널 챗봇 API
  - Claude/GPT Vision API (이미지 분석 + 카피 생성)
  - Cloudinary 또는 Canvas API (이미지 합성)
  - Supabase (유저/크레딧 관리)
  - Stripe/토스페이먼츠 (결제)
