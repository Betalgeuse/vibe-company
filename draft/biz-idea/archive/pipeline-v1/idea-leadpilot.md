## 🎯 고객 가치 (가장 중요)

### 한 문장 정의
> "이걸 쓰면 **B2B 사업자/에이전시 대표**가 **영업 시간 40시간/월**을 **절약**하면서 **매일 50개 타겟 리드**를 얻는다"

### 고객이 얻는 구체적 결과
1. **Before:** 
   - 링크드인/구글 수동 검색으로 하루 2시간 리드 찾기
   - 콜드 메일 수동 발송, 응답률 1-2%
   - 영업사원 채용하면 월 400만원+
2. **After:** 
   - 매일 아침 50개 타겟 리드 리스트 자동 도착
   - 개인화된 콜드 이메일 자동 발송
   - 응답 오면 슬랙 알림
3. **절약/획득:** 월 400만원 인건비 절감 + 하루 2시간 × 20일 = **40시간/월**

## 1. Target & Pain Point

### Target
- **1차:** 첫 B2B 세일즈 시작하는 사업자
  - 에이전시 막 시작 (직원 1-3명)
  - 프리랜서 → 에이전시 전환 중
  - B2B SaaS 프리시드~시드
- **2차:** 마케팅 에이전시 대표 (성장 중)
- **3차:** 프리랜서 컨설턴트 (고가 서비스)

### Pain Point
| 문제 | 비용 |
|------|------|
| 리드 찾기 = 수동 노가다 | 하루 2시간 × 20일 = 40시간/월 |
| 콜드 아웃리치 = 귀찮음 | 응답률 낮아 포기 |
| 영업사원 채용 = 비쌈 | 월 400만원+ (경력자) |

### 지불 의향
- 영업사원 1명(400만원)보다 싸면서 더 많은 리드 → **ROI 명확**
- B2B 계약 1건 = 수천만원, 리드 10개 중 1개만 성사해도 원금 회수

## 2. 서비스 구조

### Starter (₩100만/월)
- ICP 기반 리드 50개/일 수집
- 콜드 이메일 자동 발송 (고객 승인 후)
- 응답 시 슬랙/카톡 알림
- 월 1회 성과 리포트

### Growth (₩250만/월)
- Starter 전체 +
- 전담 최적화 (A/B 테스트)
- 월 1회 전략 미팅
- 멀티 채널 (이메일 + LinkedIn DM)

### 초기 세팅비
- ₩200만 (1회) - ICP 정의 + 템플릿 작성 + 워밍업

## 3. 핵심 기능 & 차별점

### Core Function
**"매일 50개 타겟 리드 + 자동 아웃리치"**
1. 고객이 ICP(Ideal Customer Profile) 입력
2. 시스템이 링크드인/웹에서 매칭되는 리드 50개/일 수집
3. 개인화된 콜드 이메일 자동 발송 (고객 승인 후)
4. 응답 오면 슬랙/카톡 알림

### Differentiation
| 경쟁사 | LeadPilot |
|--------|-----------|
| Apollo/ZoomInfo: 셀프서비스, 리스트만 제공 | **리스트 + 아웃리치까지 대행** |
| Clay: 강력하지만 세팅 복잡 | **세팅 완료 상태로 제공** |
| 아웃소싱 SDR: 월 500만원+ | **월 100-250만원, 더 일관된 퀄리티** |

## 5. Competition Check

### Direct Competitors
- **Apollo.io:** 리드 DB + 이메일 자동화 (셀프서비스, 월 $79+)
- **Clay:** 리드 enrichment 도구 (파워유저 대상)
- **Lemlist:** 콜드 이메일 자동화 (도구만 제공)
- **아웃소싱 SDR 대행사:** 월 500만원+

### Why They Won't Crush You
1. **Apollo/Clay = 도구 제공, 운영은 고객 몫** → LeadPilot = 완성형 서비스
2. **SDR 대행사 = 비쌈 + 품질 들쭉날쭉** → LeadPilot = 시스템화된 일관성
3. **한국 B2B 시장 특화** = 글로벌 도구들 한국어/한국 기업 데이터 부족

## 6. Growth Strategy

- **콘텐츠:** "우리가 1달간 500개 리드를 찾은 방법" 케이스 스터디

1. **무료 진단:** "현재 리드 파이프라인 분석"
