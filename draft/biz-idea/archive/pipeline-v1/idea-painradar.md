## 1. The "Micro" Opportunity
> 상혁님의 파이프라인(Hunter→Opus→Codex)을 SaaS로 상품화. Reddit에서 Pain Point 자동 추출 → 사업 아이디어 리포트 생성.

- **Target:** PM, 마케터, 인디해커, VC (시장 조사 필요한 사람)
- **Pain Point:** 
  - Reddit 수동 모니터링 시간 낭비
  - Pain Point 찾아도 정리 안 됨
  - 아이디어 검증 방법 모름
- **TAM:** Indie Hackers 10만+, r/microsaas 12만+, PM/마케터 수백만

## 2. Product Spec (MVP)

### 멀티 에이전트 파이프라인 (핵심)

```
[User 설정]          [Agent 1: Collector]     [Agent 2: Analyzer]     [Agent 3: Reporter]
키워드/서브레딧 →    Reddit 크롤링 →           Pain Point 추출 →        아이디어 리포트
                    (upvote 10+, 댓글 5+)     분류/점수화               + 원본 링크
```

### 가격 플랜
| 플랜 | 내용 | 가격 |
|------|------|------|
| **Free** | 3개 키워드, 주 1회 리포트, 7일 | $0 |
| **Pro** | 10개 키워드, 일 1회, 30일, Slack | $29/월 |
| **Team** | 무제한, 실시간, API, Notion 연동 | $99/월 |

### Tech Stack
- n8n (멀티 에이전트 오케스트레이션)
- Reddit 공식 API (ToS 준수)
- OpenAI API (분석/생성)
- Supabase (DB/Auth)
- Next.js (대시보드)
- Resend (이메일)
