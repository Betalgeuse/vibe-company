## 2. Product Spec (MVP)

- **Core Function:** 오디오/영상 업로드 → AI가 SEO 최적화된 블로그 글 + 요약 카드 자동 생성
- **Differentiation:** 
  - 한국어 구어체 → 문어체 변환 특화
  - 네이버 블로그 SEO 최적화 (키워드 밀도, 소제목 구조)
  - 브런치/티스토리 원클릭 발행 연동
- **Tech Stack:** 
  - Next.js + Supabase + Vercel
  - Whisper.cpp (로컬 STT, 비용 절감)
  - Claude/GPT API (구어체→문어체 변환)

## 4. Why This Works for YOU?

- **Low Maintenance:** 
  - Serverless 구조 (Vercel + Supabase)
  - 영상 편집 없음 → 인프라 비용 최소화
  - Whisper.cpp 로컬 실행으로 API 비용 절감
- **Scalability:** 
  - 유저당 처리 비용 약 ₩500 (10분 오디오 기준)
  - MRR ₩1,000만 달성 시에도 서버비 ₩50만 이하
- **Viral:** 
  - 생성된 글에 서비스 워터마크 삽입
  - "이 글은 Audio2Blog로 3분 만에 작성되었습니다" → 궁금증 유발
