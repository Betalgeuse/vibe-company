## 2. Product Spec (MVP)
- **Core Function:** 스마트폰으로 온도 입력 → 자동 일지 생성 → PDF 리포트 다운로드 (위생 점검용)
- **Differentiation:** 
  - 기존: 복잡한 ERP형 (사장님 혼자 못 씀)
  - 우리: 카카오톡 챗봇처럼 단순 입력, 5초 완료
- **Tech Stack:** 
  - Next.js + Supabase (DB/Auth)
  - Vercel (호스팅)
  - 토스페이먼츠 (결제)
  - PWA (앱스토어 심사 없이 모바일 앱처럼)
