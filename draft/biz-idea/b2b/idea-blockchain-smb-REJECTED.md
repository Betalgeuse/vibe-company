## 검토된 기회

### 1. 토큰 기반 로열티/멤버십 플랫폼

**ICP**
- Target: 점장/지배인 @ 독립 카페/소매점 (5-30명)
- ACV: $6K-10K/년
- TAM: $20B+ (주장) → $50-100M (현실)

**REJECT 사유**

1. **경쟁 압도적 열세**
   - Square Loyalty: $45-105/월 (우리 $500+/월의 1/10)
   - Toast: 무료 통합 제공
   - 17배 가격 차이 정당화 불가

2. **Unit Economics 붕괴**
   - 제시: LTV:CAC = 24:1
   - 현실: LTV:CAC = 1.5-2:1
   - CAC 과소평가 (4-8배)
   - LTV 과대평가 (40-60% 낮음)

3. **블록체인 필요성 없음**
   - "토큰 투명성" = 고객이 원하지 않는 기능
   - 복잡성만 증가, 가치 증가 없음
   - DB로 충분히 해결 가능한 문제

4. **고객 ROI 미검증**
   - $500/월 지출 → 어떤 수익 증가?
   - 사례 연구 0개
   - 고객 인터뷰 0명

## 핵심 Lessons Learned

### 1. 블록체인 ≠ 자동 가치

> "블록체인이라서" 돈을 내는 SMB는 없다.
> 문제 해결이 먼저, 기술은 수단일 뿐.

### 2. SMB 가격 민감도

- SMB는 $500/월도 큰 결정
- 경쟁사가 $50/월이면 10배 비싼 이유 필요
- "더 좋아서"가 아니라 "없으면 안 되는" 기능 필요

### 3. Unit Economics 현실 검증

- Founder 인건비 = CAC에 포함
- SMB Churn = 월 8-12% (높음)
- 낙관적 가정 금지
