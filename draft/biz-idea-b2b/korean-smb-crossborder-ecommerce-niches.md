## 📊 Executive Summary

- 크로스보더 전자상거래 B2B 시장 규모: $39.3T (2023) → $62.2T (2030 예상)

#### **#1: HS Code AI 자동분류 SaaS**

**📌 Pain Point:**
- 75%의 SMB가 HS Code 분류 compliance 어려움 호소
- 잘못된 분류 시 평균 $10,000+ 벌금, 배송 지연 1-2주
- 수작업 분류: 제품당 평균 15-30분 소요

**💡 솔루션:**
- AI/ML 기반 자동 HS Code 분류 (Avalara 대기업용을 SMB용으로 단순화)
- Product description + image → 95%+ 정확도 분류
- Shopify/WooCommerce 플러그인으로 원클릭 통합

**🎯 ICP (Ideal Customer Profile):**
| 항목 | 내용 |
|------|------|
| **🌍 타겟 지역** | US, EU, AU 영어권 + 한국 중소 쇼핑몰 (글로벌 판매) |
| **타겟 고객** | 연매출 10억~100억원 중소 이커머스 (월 50-500건 해외 배송) |
| **회사/팀 규모** | 1-10인 (CEO + 운영 1-2명) |
| **연매출/예산** | 연매출 $100K-$2M, SW 예산 $100-500/mo |
| **Pain 발생 빈도** | 신제품 등록 시마다 (주 5-20회) |
| **현재 해결 방법** | Excel 수작업, 포워딩 업체 의존 ($50/건), 구글링 |
| **지불 의향 근거** | 벌금 1회($10K) > 연간 구독료($2.4K) - ROI 명확 |
| **도달 채널** | Shopify App Store, Reddit r/ecommerce, Facebook 이커머스 그룹 |

**💰 가격 모델:**
- **Starter:** $49/mo (월 100건, 기본 API)
- **Growth:** $149/mo (월 500건, Shopify 통합, 이메일 지원)
- **Pro:** $349/mo (무제한, 우선 지원, Bulk API)

**📈 ARR 계산:**
- Year 1: 50 customers × $149 avg = **$7,450 MRR → $89K ARR** ✅
- Year 2: 150 customers × $149 avg = **$22,350 MRR → $268K ARR** 🎯
- Target: 고객 100명만 확보해도 **$178K ARR** 달성

**🤖 자동화 가능성: 10/10**
- AI 모델: OpenAI GPT-4V + Fine-tuning (HS Code 데이터셋)
- API 통합: Shopify/WooCommerce Webhook 자동 처리
- 데이터 업데이트: WCO Harmonized System 크롤링 (월 1회 자동)
- 고객 응대: Intercom chatbot (95% 자동 응답)

**1인 운영 복잡도: ⭐⭐ (Medium)**
- 초기 투자: AI 모델 훈련 ($5K-10K), 플러그인 개발 (2-3개월)
- 운영 비용: API 비용 ($500-2K/mo), 서버 ($200/mo)
- **월 업무량:** 10-15시간 (모델 개선 5h, 고객 지원 5h, 마케팅 5h)

**경쟁 현황:**
- Avalara: 대기업용, 복잡, 비쌈 ($1K+/mo)
- Zonos Classify: 개발자 중심, SMB 친화적 아님
- **갭:** $50-200/mo 가격대, Shopify 네이티브 통합, 한국어 지원 부재

#### **#2: Landed Cost 실시간 계산 SaaS**

**📌 Pain Point:**
- 82%의 이커머스가 landed cost 과소평가 → 평균 18% 마진 손실
- 배송비, 관세, VAT, 보험, 통관 수수료 등 10개+ 변수 추적 불가
- 엑셀 수작업: 오류율 35%, 계산 시간 SKU당 20분

**💡 솔루션:**
- 실시간 Landed Cost Calculator (배송지 + 제품 정보 → 총 비용 자동 산출)
- Shopify 결제 페이지에서 고객에게 "총 착불 비용" 투명하게 표시
- 환율, 관세율 자동 업데이트 (API 연동)

**🎯 ICP:**
| 항목 | 내용 |
|------|------|
| **🌍 타겟 지역** | Global (특히 EU, US, AU 판매하는 SMB) |
| **타겟 고객** | 크로스보더 직구 쇼핑몰 (K-뷰티, K-패션, 전자제품) |
| **회사/팀 규모** | 1-5인 |
| **연매출/예산** | $200K-$3M revenue, $100-300/mo SW budget |
| **Pain 발생 빈도** | 매 주문마다 (일 10-100건) |
| **현재 해결 방법** | 엑셀 템플릿, 포워딩 업체 견적 ($30/건), 추정값 |
| **지불 의향 근거** | 마진 손실 18% 방지 = 월 $5K+ 절약 (연매출 $500K 기준) |
| **도달 채널** | Shopify App Store, Product Hunt, Indie Hackers, LinkedIn (이커머스 그룹) |

**💰 가격 모델:**
- **Free:** 월 10건 계산 (리드 생성용)
- **Starter:** $79/mo (월 100건, 기본 API, 3개국)
- **Pro:** $199/mo (월 500건, 무제한 국가, Checkout 통합)
- **Enterprise:** $499/mo (무제한, White-label, API)

**📈 ARR 계산:**
- Year 1: 60 customers × $199 avg = **$11,940 MRR → $143K ARR** ✅
- Year 2: 120 customers × $199 avg = **$23,880 MRR → $286K ARR** 🎯

**🤖 자동화 가능성: 9/10**
- 데이터 소스: Avalara API, WCO Tariff DB, Freightos API (배송비)
- 계산 엔진: 자동화된 공식 (국가별 룰 엔진)
- 통합: Shopify/WooCommerce App (결제 페이지 위젯)
- **업데이트:** 관세율 변경 자동 크롤링 (주 1회)

**1인 운영 복잡도: ⭐⭐ (Medium)**
- 초기 투자: API 통합 개발 ($3K-5K), 2개월
- 운영 비용: 외부 API ($300-1K/mo), 서버 ($150/mo)
- **월 업무량:** 12-18시간 (데이터 검증 6h, 지원 6h, 개선 6h)

**경쟁 현황:**
- Dutify: API 중심, SMB 친화적 아님
- Parabola: 복잡한 워크플로우 빌더 (learning curve 높음)
- **갭:** Shopify 네이티브, 한 번에 설정 끝나는 간편함

#### **#3: 다국어 SEO 자동 최적화 SaaS**

**📌 Pain Point:**
- 76%의 소비자가 자국어 콘텐츠 선호하지만, 기계 번역 품질 불만족률 60%+
- SEO 최적화 없는 번역 → 검색 노출 0에 가까움
- 현지 키워드 조사 수작업: 언어당 20시간+ 소요

**💡 솔루션:**
- AI 기반 SEO 최적화 번역 (단순 번역 X, 검색 의도 반영 O)
- 현지 검색어 자동 추출 + 메타태그 최적화 + hreflang 자동 설정
- Shopify/WooCommerce 플러그인 (버튼 클릭 한 번에 10개 언어 SEO 완료)

**🎯 ICP:**
| 항목 | 내용 |
|------|------|
| **🌍 타겟 지역** | Global (US, EU, LATAM, APAC 진출하는 브랜드) |
| **타겟 고객** | D2C 브랜드 (K-뷰티, 패션, 홈인테리어 등) |
| **회사/팀 규모** | 3-15인 (마케팅 1-2명) |
| **연매출/예산** | $500K-$5M revenue, $200-800/mo marketing SW budget |
| **Pain 발생 빈도** | 신제품 출시 시 (월 5-20개 SKU), 콘텐츠 업데이트 (주 1-2회) |
| **현재 해결 방법** | Weglot($100/mo) + 수작업 SEO, 번역 에이전시 ($500-2K/건) |
| **지불 의향 근거** | 오가닉 트래픽 3배 증가 = 월 $10K+ 매출 증가 (CAC 절감) |
| **도달 채널** | Product Hunt, Shopify App Store, r/ecommerce, LinkedIn ads |

**💰 가격 모델:**
- **Starter:** $99/mo (3개 언어, 100 pages, 기본 SEO)
- **Growth:** $249/mo (10개 언어, 500 pages, 고급 SEO, A/B 테스트)
- **Scale:** $599/mo (무제한 언어/페이지, API, 전담 지원)

**📈 ARR 계산:**
- Year 1: 40 customers × $249 avg = **$9,960 MRR → $119K ARR** ⚠️ (목표 미달)
- Year 2: 80 customers × $299 avg = **$23,920 MRR → $287K ARR** ✅
- **전략:** Year 1에 고객 60명 확보 → **$179K ARR** 달성 가능

**🤖 자동화 가능성: 9/10**
- 번역: OpenAI GPT-4 + 각 언어별 SEO 프롬프트 최적화
- 키워드 조사: Ahrefs API, SEMrush API (자동 추출)
- 메타태그: 템플릿 기반 자동 생성
- 크롤링: Screaming Frog API 통합 (주간 SEO 리포트)

**1인 운영 복잡도: ⭐⭐⭐ (Medium-High)**
- 초기 투자: AI 프롬프트 엔지니어링 ($2K-4K), 플러그인 개발 (2-3개월)
- 운영 비용: OpenAI API ($800-2K/mo), SEO 도구 API ($300/mo)
- **월 업무량:** 18-25시간 (프롬프트 개선 8h, 지원 7h, 마케팅 10h)

**경쟁 현황:**
- Weglot, Smartcat: 번역만 제공 (SEO 최적화 부족)
- LILT, Phrase: 대기업용, 복잡함
- **갭:** SEO + 번역 통합, SMB 가격대 ($100-300/mo)
