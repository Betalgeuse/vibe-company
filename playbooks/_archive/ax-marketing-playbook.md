## 🏠 luerre AI 연결

> **기준 문서**: [company.md](./company.md)

### Vision
**Make people do what they love**
- 고객이 비즈니스에만 집중할 수 있게
- 개발자가 코딩에만 집중할 수 있게

### 이 직무의 Mission
**MBA 비기술 창업자에게 luerre AI를 알리기** - 타겟 고객(MBA 출신 비기술 창업자)에게 외주 개발의 새로운 대안을 알리고, 영업 가능한 리드를 생성합니다.

### Core Values 실천

| Core Value | 이 직무에서의 실천 |
|------------|-------------------|
| **No ego, Just impact** | 팔로워 수보다 Qualified Lead 수 |
| **Don't (just) get shit done** | 의미 없는 콘텐츠 양산 X, 타겟에게 도움되는 것만 |

### Problem 해결 기여

| company.md Problem | AX Marketing 기여 |
|-------------------|------------------|
| Scope Creep | 콘텐츠로 Scope 관리의 중요성 교육 |
| 불투명한 진행 | 투명한 외주 개발의 가치 전달 |
| 커뮤니케이션 오버헤드 | 사전 교육으로 고객 기대치 정렬 |

### 타겟 고객 (company.md 기준)
- **MBA 출신 또는 비즈니스 배경의 창업자**
- 명확한 비즈니스 아이디어와 시장 검증 완료
- 기술적 배경은 부족하지만 빠르게 MVP를 만들고 싶은 사람

### 팀 철학
우리는 이런 사람을 환영합니다:
- 평생 공부만 했는데 스타트업 씬에 도전해보고 싶은 사람
- The brave dropout - 용기 있게 선택했고, 꾸준히 공부하는 사람
- 은퇴했지만 7살의 마인드셋으로 배우고 싶은 사람

## 🎯 The One Number: Qualified Leads / 월

| 항목 | 내용 |
|------|------|
| **목표** | 20개/월 |
| **왜?** | Marketing의 본질 = 영업 가능한 리드 생성 |
| **측정** | ICP 조건 충족하는 인바운드 리드 |

**ICP (Ideal Customer Profile)**:
- MBA 출신 또는 비기술 창업자
- 예산 있음 (최소 ₩5,000,000)
- 3개월 내 프로젝트 시작 의향
- 웹/앱 개발 필요

**보조 지표**:
- 콘텐츠 발행 수 (주 2개 이상)
- 웹사이트 트래픽 (월 1,000+)
- 리드당 비용 (CAC) < ₩100,000
- 이메일 오픈율 > 25%

## 🎯 ICP (Ideal Customer Profile) 상세

### 타겟 페르소나

```
┌─────────────────────────────────────────────────────┐
│  Primary ICP: MBA 출신 비기술 창업자                 │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Demographics:                                      │
│  - 나이: 28-45세                                    │
│  - 학력: MBA 또는 경영/비즈니스 전공                 │
│  - 위치: 서울/수도권, 미국 주요 도시                 │
│                                                     │
│  Psychographics:                                    │
│  - 비즈니스 아이디어는 있지만 개발 못함              │
│  - 외주 개발 경험 있음 (대부분 실패)                 │
│  - "투명한 프로세스"에 목마름                        │
│  - 시간 > 돈 (빠른 런칭 원함)                       │
│                                                     │
│  Pain Points:                                       │
│  - "개발사 믿을 수 있나?"                           │
│  - "예산이 끝없이 늘어남"                           │
│  - "언제 끝나는지 모름"                             │
│  - "개발자와 소통이 너무 어려움"                     │
│                                                     │
│  Where they hang out:                               │
│  - LinkedIn                                         │
│  - 스타트업 커뮤니티 (디스콰이엇, 온오프믹스)         │
│  - MBA 동문 네트워크                                │
│  - 창업 관련 뉴스레터                               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### ICP 자격 체크리스트

```markdown
## 리드 자격 검증

### 필수 조건 (모두 충족)
- [ ] 비기술 배경 (개발자 아님)
- [ ] 웹/앱 개발 필요
- [ ] 예산 ₩5,000,000 이상
- [ ] 3개월 내 시작 의향

### 가산점
- [ ] MBA 출신
- [ ] 외주 개발 실패 경험
- [ ] 명확한 비즈니스 모델
- [ ] 투자 유치 완료

### 점수
- 필수 4개 + 가산점 2개 이상 → Hot Lead 🔥
- 필수 4개 + 가산점 1개 → Warm Lead
- 필수 4개만 → Lead
- 필수 미충족 → Disqualify
```

## 📝 콘텐츠 마케팅

### 콘텐츠 종류 및 목적

| 종류 | 목적 | 주기 | 채널 |
|------|------|------|------|
| **블로그** | SEO, 교육 | 주 1회 | 웹사이트 |
| **케이스 스터디** | 신뢰 구축 | 월 1회 | 웹사이트, LinkedIn |
| **LinkedIn 포스트** | 인지도 | 주 3회 | LinkedIn |
| **뉴스레터** | 리드 육성 | 격주 | 이메일 |
| **웨비나** | 리드 생성 | 월 1회 | Zoom |

### 블로그 주제 아이디어

**Pain Point 기반:**
- "외주 개발 실패하는 5가지 이유 (그리고 해결책)"
- "MVP 개발, 얼마나 걸리나요? 현실적인 가이드"
- "개발사 선택 시 반드시 물어봐야 할 10가지 질문"
- "Scope Creep이란? 프로젝트를 망치는 주범"

**Solution 기반:**
- "AI가 바꾸는 외주 개발의 미래"
- "투명한 외주 개발이 가능한 이유"
- "MBA 창업자를 위한 기술 없이 MVP 만들기"

**케이스 스터디:**
- "[고객사] 3주 만에 MVP 런칭한 비결"
- "Scope Creep 0건, 예산 초과 0건의 비밀"

```markdown

## 해결책
[luerre AI의 접근 방식]

[케이스 스터디 요약]

[CTA 버튼: 무료 상담 신청하기]

## 📊 SEO/SEM

### SEO 전략

**타겟 키워드:**

| 키워드 | 검색량 | 난이도 | 우선순위 |
|--------|--------|--------|----------|
| 외주 개발 | 높음 | 높음 | P2 |
| MVP 개발 | 중간 | 중간 | P1 |
| 앱 개발 외주 | 중간 | 중간 | P1 |
| 스타트업 개발 | 낮음 | 낮음 | P0 |
| 웹 개발 비용 | 중간 | 높음 | P2 |

**콘텐츠-키워드 매핑:**
- "MVP 개발" → MVP 가이드 블로그
- "외주 개발 비용" → 견적 가이드 블로그
- "앱 개발 외주" → 케이스 스터디

### SEM (유료 광고)

**채널 우선순위:**
1. **Google Ads** - 검색 의도 높음
2. **LinkedIn Ads** - MBA 타겟팅 정확
3. **Facebook/Instagram** - 비용 효율

**Google Ads 캠페인 구조:**
```
Campaign: luerre AI - MVP 개발
├── Ad Group: MVP 개발
│   ├── Keyword: mvp 개발
│   ├── Keyword: mvp 개발 비용
│   └── Keyword: mvp 개발 업체
├── Ad Group: 외주 개발
│   ├── Keyword: 외주 개발
│   └── Keyword: 앱 외주 개발
└── Ad Group: 스타트업 개발
    ├── Keyword: 스타트업 개발
    └── Keyword: 창업 개발
