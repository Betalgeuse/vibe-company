# 디자인 레퍼런스 - 실행 전 체크리스트

## ✅ 검색 전 확인

### 목표 정의
- [ ] UI 종류 명확히 정의 (로그인/대시보드/랜딩 등)
- [ ] 플랫폼 확정 (웹/모바일/데스크톱)
- [ ] 원하는 분위기 키워드 3개 선정
- [ ] 타겟 사용자 정의

### 검색 키워드 준비
```
내 검색 키워드:
1. ________________
2. ________________
3. ________________
```

## ✅ 검색 중 체크

### 필수 확인 사이트
- [ ] Mobbin에서 실제 앱 패턴 확인
- [ ] Dribbble에서 시각적 영감 수집
- [ ] 1개 이상 전문 사이트 확인 (Landbook, Godly 등)

### 수집 기준
- [ ] 최소 3개 이상 레퍼런스 확보
- [ ] 각 레퍼런스의 "참고할 점" 메모
- [ ] 최근 1-2년 이내 작업물인지 확인
