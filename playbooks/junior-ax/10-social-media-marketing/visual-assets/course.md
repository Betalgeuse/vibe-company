# Course: Visual Assets 실행 가이드

## 📚 Step-by-Step (60분 완성)

### Step 1: OG Image 제작 (15분)

#### Option A: OG Image Playground (가장 빠름)

```typescript
// https://og-playground.vercel.app/

1. Template 선택 (Simple 또는 Product)
2. 텍스트 입력:
   - Title: [제품명]
   - Description: [Tagline]
3. 이미지 업로드: 로고 or 스크린샷
4. 색상 조정: 브랜드 색상
5. Export PNG (1200x630)
```

#### Option B: Canva (커스텀 디자인)

```
1. "OG Image" 템플릿 검색
2. 레이아웃 선택
3. 요소 추가:
   - 배경 (브랜드 색상 또는 그라데이션)
   - 로고 (좌측 상단)
   - 제품명 (큰 폰트, 중앙)
   - Tagline (작은 폰트, 하단)
   - 스크린샷 or 일러스트 (우측)
4. Download PNG
```

**체크리스트:**
- [ ] 1200x630px
- [ ] 제품명 가독성 (작은 화면에서도)
- [ ] 브랜드 색상 사용
- [ ] 텍스트 너무 많지 않음 (5-10 단어)

### Step 2: Screenshots 촬영 (20분)

#### 준비 (5분)

```
1. 테스트 데이터 준비
   - 실제 사용자 데이터 X
   - 예시 데이터로 대체
   - 민감 정보 제거

2. 브라우저 설정
   - Zoom 100%
   - 해상도 1920x1080
   - Extension 숨기기
   - 개발자 도구 닫기
```

#### 촬영 (15분)

**Screenshot 1: Hero (메인 화면)**
```
- 첫 인상이 중요
- 전체 UI 보여주기
- 로그인 후 Dashboard 또는 Main Page
```

**Screenshot 2-3: 핵심 기능**
```
- 가장 중요한 기능 2개
- 사용 중인 화면 (액션 포함)
- Before/After 또는 Process 강조
```

**Screenshot 4: 결과 화면**
```
- 완료 화면
- 데이터 시각화 (그래프, 차트)
- 성취감 주는 화면
```

**Screenshot 5: Mobile (선택)**
```
- 브라우저 DevTools → Responsive
- iPhone 13 사이즈
- 핵심 화면 1개
```

#### 후처리 (5분)

```
도구: Figma 또는 Photoshop

1. 리사이즈: 1920x1080
2. 둥근 모서리 (선택): 8-16px radius
3. 그림자 추가 (선택): Drop shadow
4. 배경 제거 또는 색상 변경
5. Export PNG
```

### Step 3: Demo Video 제작 (20분)

#### 스크립트 작성 (5분)

```
[0-5초] Hook
"Tired of [문제]?"

[5-10초] Solution
"Meet [제품]. It [핵심 가치]."

[10-40초] Demo
1. [액션 1]: [결과]
2. [액션 2]: [결과]
3. [액션 3]: [결과]

[40-50초] Result
"In 30 seconds, we [결과]."

[50-60초] CTA
"Try it free at [URL]"
```

#### 녹화 (10분)

**Loom 사용 (추천):**
```
1. Loom 확장 프로그램 설치
2. "Screen + Camera" 선택 (또는 Screen only)
3. 녹화 시작
4. 스크립트대로 시연
5. 한 번에 녹화 (실수해도 OK, 자연스러움)
6. 녹화 종료
```

**주의사항:**
- [ ] 마우스 움직임 천천히
- [ ] 중요한 부분 강조 (클릭, 입력)
- [ ] 음성 or 자막 (최소 자막 필수)
- [ ] 배경 음악 (선택, 저작권 Free)

#### 편집 (5분)

```
Loom 내장 편집:
1. Trim (앞뒤 여백 제거)
2. 자막 추가 (Settings → Captions)
3. CTA 추가 (마지막 프레임)
4. Download MP4
```

### Step 4: Social Post Images (5분)

#### Canva Quick Templates

**Feature Highlight:**
```

2. 배경: 브랜드 색상
3. 텍스트: "[기능명]" + "✨ [이점]"

```

**User Testimonial:**
```
1. Template: "Quote Post"
2. 배경: Gradient
3. 인용문: "[피드백]"
```

```
1. Template: "Statistics Post"
2. 큰 숫자: "[100+]"
3. 설명: "users in 24h"
4. 아이콘 추가
```

## 💡 Pro Tips

### 1. 일관된 스타일
```
모든 에셋에 동일하게 적용:
- 폰트 2개 (Heading, Body)
- 색상 팔레트 (Primary, Secondary)
- Border radius (8px or 16px)
- 그림자 스타일
```

### 2. 텍스트 최소화
```
Visual은 3초 안에 이해 가능해야 함

❌ 긴 설명문
✅ 핵심 키워드 + 스크린샷
```

### 3. Retina 준비
```
최소 해상도:
- OG Image: 1200x630 (2x)
- Screenshots: 1920x1080 (2x)
- Social: 1080x1080 (2x)
```

### 4. 재사용 가능
```
템플릿 저장:
- Canva Template 저장
- Figma Component 저장
- Screenshot 위치 기억

다음 포스트에 재사용
```
