# Checklist: Engagement Strategy

## ✅ Pre-Execution

### 준비물
- [ ] 브랜딩 완료 (톤앤매너)
- [ ] ChatGPT/Claude 접속
- [ ] Google Docs 또는 Notion 준비
- [ ] 모든 플랫폼 계정 로그인 확인

### 시간 확보
- [ ] 45분 타임박스 설정

## ✅ During Execution

### Step 1: 템플릿 생성 (30분)
**ChatGPT 생성 (20분)**
- [ ] 프롬프트 작성 (제품 정보 입력)
- [ ] 6가지 유형 템플릿 요청
- [ ] 결과 검토 및 수정

**유형별 템플릿 (각 3-5개)**
- [ ] Positive Feedback (긍정 피드백)
- [ ] Questions (질문)
- [ ] Bug Reports (버그 제보)
- [ ] Feature Requests (기능 요청)
- [ ] Negative Feedback (부정 피드백)
- [ ] Collaboration (협업 제안)

**정리 (10분)**
- [ ] Google Docs 또는 Notion에 정리
- [ ] 유형별 분류
- [ ] 변수 표시 ([제품명], [기능명])
- [ ] 예시 추가

### Step 2: 시스템 구축 (10분)
**알림 설정**
- [ ] Twitter/X: Mentions, Replies, DM (Push ON)
- [ ] LinkedIn: Comments, Mentions, Messages (Push ON)
- [ ] ProductHunt: Comments, Mentions (Push ON)
- [ ] Reddit: Replies, Mentions (Push ON)
- [ ] 모바일 앱 설치 확인

**템플릿 접근**
- [ ] Google Docs 북마크 또는
- [ ] Notion Quick Access 또는
- [ ] Text Expander 설정 (선택)
- [ ] 모바일에서도 접근 가능 확인

**미응답 방지**
- [ ] 스프레드시트 템플릿 생성 또는
- [ ] Notion Database 설정
- [ ] 30분 알림 설정

### Step 3: 프로세스 정의 (5분)
- [ ] 30분 내 응답 플로우 문서화
- [ ] 우선순위 시스템 정의
  - [ ] 🔴 Urgent
  - [ ] 🟡 Medium
  - [ ] 🟢 Easy
- [ ] Escalation 프로세스 정의

## ✅ Post-Execution

### 템플릿 검증
- [ ] 총 20개 이상 템플릿?
- [ ] 6가지 유형 모두 커버?
- [ ] 브랜드 톤앤매너 일치?
- [ ] 변수 명확히 표시?

### 시스템 테스트
- [ ] 알림 작동 확인 (테스트 댓글)
- [ ] 템플릿 빠른 접근 가능?
- [ ] 미응답 추적 시스템 작동?

### 팀 공유 (있으면)
- [ ] 템플릿 문서 공유
- [ ] 응답 프로세스 브리핑
- [ ] 역할 분담 (플랫폼별)

## ✅ Launch Week Preparation

### Day -1
- [ ] 템플릿 최종 검토
- [ ] 시스템 테스트
- [ ] 모바일 알림 확인

### Day 0 (Launch Day)
- [ ] 알림 Volume 최대
- [ ] 템플릿 항상 열어두기
- [ ] 30분 내 응답 목표 재확인

### Week 1
- [ ] 매일 응답률 측정
- [ ] 템플릿 개선 (필요 시)
- [ ] 미응답 0개 유지

## ✅ Quality Check

### 응답 품질
- [ ] 진정성 있는가? (자동화 느낌 X)
- [ ] 가치 제공하는가? (단순 "Thanks" X)
- [ ] Engagement 유도하는가? (질문 포함)
- [ ] 브랜드 톤 일치하는가?

### 응답 속도
- [ ] 평균 응답 시간 30분 이내?
- [ ] 응답률 100%?
- [ ] Urgent 즉시 응답?

### 템플릿 커버리지
- [ ] 모든 상황 대응 가능?
- [ ] 예상하지 못한 상황 발생 시 어떻게?
- [ ] 팀원도 사용 가능? (있으면)

## ✅ Real-time Response Checklist

### 댓글 받았을 때
- [ ] 5분 내: 읽기 + 분류 (🔴🟡🟢)
- [ ] 10분 내: 템플릿 선택
- [ ] 15분 내: 커스터마이즈
- [ ] 20분 내: 검토
- [ ] 30분 내: 발송 ✅

### 응답 후
- [ ] 미응답 리스트에서 체크
- [ ] 추가 질문 있으면 알림 설정
- [ ] Positive → 스크린샷 (Testimonial로 활용)

### 하루 종료 시
- [ ] 미응답 0개 확인
- [ ] 내일 Follow-up 필요한 것 표시
- [ ] 응답률 계산 (100% 목표)

## 📊 Success Metrics

### 즉시 측정 (매일)
- [ ] 응답률: 100%
- [ ] 평균 응답 시간: 30분 이내
- [ ] 미응답: 0개

### Week 1 측정
- [ ] 총 댓글/DM 수: [숫자]
- [ ] 응답한 개수: [숫자]
- [ ] 응답률: [%]
- [ ] 평균 응답 시간: [분]
- [ ] Positive sentiment: [%]

### FCSR Tracking
```
Engagement FCSR = 
  (30분 내 응답한 댓글 수) / 
  (전체 댓글 수) × 100%

Target: 80%+

24/30 댓글 30분 내 응답 → 80% ✅
```

### Template Effectiveness
```
1주일 후:
- 가장 많이 쓴 템플릿 Top 5
- 개선 필요한 템플릿 파악
- 새로 필요한 상황 추가
```

## ⚠️ Common Pitfalls

### 응답 실수
- [ ] 방어적 응답 금지 (부정 피드백에)
- [ ] 복붙 느낌 금지 (개인화 필요)
- [ ] 너무 긴 응답 금지 (Twitter)
- [ ] 질문 무시 금지 (모든 질문 답변)

### 시스템 실수
- [ ] 알림 끄기 금지
- [ ] 미응답 방치 금지 (24시간 경과)
- [ ] 템플릿 과의존 금지 (상황에 맞게 조정)

### 타이밍 실수
- [ ] Launch Day 늦은 응답 금지
- [ ] 부정 피드백 지연 금지
- [ ] 버그 제보 무시 금지
