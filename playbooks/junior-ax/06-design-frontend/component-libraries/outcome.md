# 컴포넌트 라이브러리 활용

## 🎯 목표
검증된 컴포넌트 라이브러리를 활용하여 **처음부터 만들지 않고** 빠르게 UI 구축

## 📊 성공 지표
- [ ] 버튼, 폼, 모달 등 직접 만들지 않음
- [ ] 일관된 디자인 시스템 유지
- [ ] 컴포넌트 조합으로 페이지 완성 시간 50% 단축

## 🔧 핵심 라이브러리

### Tier 1: 필수 (프로젝트에 최소 1개)

| 라이브러리 | 특징 | URL |
|------------|------|-----|
| **shadcn/ui** | 복사해서 쓰는 컴포넌트, 커스텀 자유도 최고 | https://ui.shadcn.com |
| **Radix UI** | 접근성 완벽, Headless | https://radix-ui.com |
| **Headless UI** | Tailwind 공식, 접근성 기반 | https://headlessui.com |

### Tier 2: 애니메이션 & 효과

| 라이브러리 | 특징 | URL |
|------------|------|-----|
| **Magic UI** | 화려한 애니메이션 컴포넌트 | https://magicui.design |
| **Aceternity UI** | 트렌디한 효과 컴포넌트 | https://ui.aceternity.com |
| **Framer Motion** | React 애니메이션 표준 | https://framer.com/motion |

### Tier 3: 완성형 템플릿

| 라이브러리 | 특징 | URL |
|------------|------|-----|
| **Tailwind UI** | 유료, 고품질 완성형 | https://tailwindui.com |
| **DaisyUI** | Tailwind 플러그인, 쉬운 사용 | https://daisyui.com |
| **Chakra UI** | 쉬운 테마 설정 | https://chakra-ui.com |

### 아이콘 & 일러스트

| 리소스 | 특징 | URL |
|--------|------|-----|
| **Lucide** | shadcn 기본 아이콘 | https://lucide.dev |
| **Heroicons** | Tailwind 공식 아이콘 | https://heroicons.com |
| **Phosphor** | 다양한 스타일 | https://phosphoricons.com |

## ✅ 완료 조건
1. 프로젝트에 1개 이상 컴포넌트 라이브러리 설정
2. 기본 컴포넌트 (Button, Input, Modal) 라이브러리에서 가져오기
3. 커스터마이징 방법 숙지
