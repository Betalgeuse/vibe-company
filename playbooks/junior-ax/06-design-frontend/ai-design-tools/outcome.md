# AI 디자인 도구 활용

## 🎯 목표
AI 기반 디자인 도구를 활용하여 **5분 안에** 프론트엔드 프로토타입 생성

## 📊 성공 지표
- [ ] 텍스트 프롬프트로 UI 컴포넌트 생성 가능
- [ ] 생성된 코드를 프로젝트에 바로 적용 가능
- [ ] 반복 수정 없이 원하는 결과물 획득 (FCSR 80%+)

## 🔧 핵심 도구

### 1. v0.dev (Vercel)
- **용도:** React + Tailwind 컴포넌트 즉시 생성
- **강점:** 코드 품질 높음, shadcn/ui 기반
- **URL:** https://v0.dev

### 2. Google Stitch
- **용도:** 자연어로 전체 앱 프로토타입 생성
- **강점:** 빠른 아이디어 검증, 인터랙티브 프로토타입
- **URL:** https://stitch.withgoogle.com

### 3. Galileo AI
- **용도:** 텍스트 → 완성된 UI 디자인
- **강점:** Figma 연동, 고품질 디자인
- **URL:** https://www.usegalileo.ai

### 4. Uizard
- **용도:** 스케치/스크린샷 → 편집 가능한 디자인
- **강점:** 손그림도 인식, 빠른 변환
- **URL:** https://uizard.io

## ✅ 완료 조건
1. 최소 2개 이상의 AI 도구 사용 경험
2. 생성된 코드로 실제 컴포넌트 구현 완료
3. 도구별 장단점 파악 및 상황별 선택 가능
