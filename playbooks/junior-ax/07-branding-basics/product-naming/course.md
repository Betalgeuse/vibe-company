# Course: Product Naming 실행 가이드

## 📚 Step-by-Step (30분 완성)

### Step 1: 제품 핵심 정리 (5분)

**질문에 답하기:**
```
1. 이 제품은 무엇을 하는가? (한 문장)
   예: "개발자를 위한 AI 코드 리뷰 도구"

2. 타겟 사용자는?
   예: "스타트업 개발팀"

3. 핵심 가치는?
   예: "코드 품질 향상, 시간 절약"

4. 연상시키고 싶은 이미지는?
   예: "빠름, 정확함, 신뢰"
```

---

### Step 2: AI로 후보 생성 (10분)

#### ChatGPT 프롬프트 (복사 붙여넣기)

```
당신은 글로벌 브랜딩 전문가입니다. 
아래 제품의 이름을 10개 제안해주세요.

[제품 정보]
- 제품: [Step 1에서 정리한 내용]
- 타겟: [타겟 사용자]
- 핵심 가치: [핵심 가치]
- 연상 이미지: [연상 이미지]

[요구사항]
1. 발음하기 쉬운 이름 (3음절 이내)
2. 의미가 연상되는 이름
3. 도메인 확보 가능성 고려
4. 글로벌하게 통용 가능

[형식]
각 이름마다:
- 이름
- 의미
- 연상 이미지
- 도메인 후보 (예: brandname.com)

10개를 표로 정리해주세요.
```

#### 예시 결과
| 이름 | 의미 | 연상 이미지 | 도메인 |
|------|------|-------------|--------|
| Codex | 코드 + 지수 | 전문적, 체계적 | codex.ai |
| Reviewly | Review + ly | 친근함, 반복 | reviewly.io |
| Lintify | Lint + ify | 깨끗함, 정리 | lintify.com |

---

### Step 3: 도메인 확인 (10분)

#### 도메인 검색 사이트
```
1차: Namecheap (https://www.namecheap.com)
2차: GoDaddy (https://www.godaddy.com)
3차: 구글 도메인 (https://domains.google)
```

#### 확인 우선순위
```
1. .com (최우선)
2. .io (Tech 스타트업)
3. .ai (AI 제품)
4. .app (앱 서비스)
5. .co (차선책)
```

#### 실행 방법
```bash
# 1. Namecheap 접속
# 2. 각 후보 이름 검색
# 3. 스프레드시트에 기록

| 이름 | .com | .io | .ai | 가격 | 상태 |
|------|------|-----|-----|------|------|
| Codex | ❌ | ✅ | ❌ | $29 | 구매 가능 |
| Reviewly | ✅ | ✅ | ✅ | $12 | 구매 가능 |
| Lintify | ✅ | ✅ | ❌ | $15 | 구매 가능 |
```

---

### Step 4: 후보 좁히기 (3분)

#### 평가 기준표
```
각 후보에 점수 부여 (1-5점):

기준:
1. 발음 쉬움 (__/5)
2. 기억하기 쉬움 (__/5)
3. 의미 연상 (__/5)
4. 도메인 확보 (__/5)
5. 로고 만들기 쉬움 (__/5)

총점: __/25
```

#### 예시
| 이름 | 발음 | 기억 | 의미 | 도메인 | 로고 | 총점 |
|------|------|------|------|--------|------|------|
| Codex | 5 | 5 | 4 | 4 | 5 | 23 |
| Reviewly | 4 | 4 | 5 | 5 | 4 | 22 |
| Lintify | 5 | 4 | 3 | 5 | 4 | 21 |

**결과:** Codex 선택!

### Step 5: 최종 검증 (5분)

#### 1. 구글 검색
```
"Codex" 검색 → 결과 확인
- 기존 유명 제품/서비스 있나?
- 부정적 의미 없나?
```

#### 2. 상표권 검색 (한국)
```
KIPRIS (https://www.kipris.or.kr)
→ 상표 검색 → "Codex" 입력
→ 동일/유사 상표 있나?
```

#### 3. SNS 계정 확인
```
Twitter: @codex
Instagram: @codex
LinkedIn: /company/codex

✅ 확보 가능하면 즉시 생성!
```

#### 4. 다국어 의미 확인
```
ChatGPT 프롬프트:

"Codex"라는 단어가 다음 언어에서 부정적 의미가 있나요?
- 영어
- 한국어
- 일본어
- 중국어
- 스페인어
- 프랑스어
```

### Step 6: 스토리 작성 (5분)

#### 템플릿
```markdown
# 제품명: Codex

## 의미
"Code"와 "Index"의 합성어. 
코드의 모든 것을 체계적으로 정리한다는 의미.

## 왜 이 이름인가?
고대 필사본 "Codex"처럼, 
개발자의 코드를 소중히 다루고 정리한다는 철학.

## 브랜드 스토리
개발자들은 매일 코드를 작성하지만, 
품질 관리는 항상 뒷전이었습니다.

Codex는 AI로 코드를 분석하고,
개발자가 더 나은 코드를 쓸 수 있도록 돕습니다.

마치 고대의 Codex가 지식을 보존했듯이,
우리는 코드 품질을 지킵니다.

## 연상 이미지
- 전문적
- 체계적
- 신뢰감
- 역사적 깊이
```

## 🎯 실전 예시

### Case Study: "Notion" 네이밍

**1. 제품 정리**
- 제품: 올인원 워크스페이스
- 타겟: 지식 근로자
- 가치: 모든 것을 한 곳에

**2. 이름 선정**
- "Notion" = "개념"
- 모든 개념/아이디어를 담는다는 의미

**3. 도메인**
- notion.so 확보 (notion.com은 비쌈)

**4. 결과**
- 발음 쉬움 ✅
- 기억 쉬움 ✅
- 의미 연상 ✅
- 브랜딩 성공 ✅

## 💡 Pro Tips

### 1. ChatGPT 프롬프트 고급 기법
```
레퍼런스 기반 프롬프트:

"다음 성공 사례들을 참고해서 이름을 지어주세요:
- Slack: 협업 도구, 짧고 강렬
- Stripe: 결제 도구, 간결하고 기술적
- Linear: 이슈 트래커, 직선적인 워크플로우

[내 제품 정보]
..."
```

### 2. 도메인 Hack 활용
```
예:
- bit.ly (Bitly)
- del.icio.us (Delicious)
- about.me

국가 도메인을 창의적으로 활용!
```

### 3. 조합 네이밍
```
방법:
1. 단어 + ly: Review → Reviewly
2. 단어 + ify: Lint → Lintify
3. 단어 + able: Share → Shareable
4. 축약: Google Drive → GDrive
```

### 4. 테스트 방법
```
"엄마 테스트":
전화로 엄마에게 제품명 설명할 수 있나?

"바 테스트":
시끄러운 바에서 친구에게 외쳐서 전달 가능한가?
```
