# 🎯 Skill 사용하기 - Outcome

## 목표

**Factory Skill을 활용하여 프롬프트 품질과 작업 효율성을 높인다.**

---

## 성공 기준

| 항목 | 기준 |
|------|------|
| Skill 실행 | `prompt-enhancer` skill을 호출하여 프롬프트 개선 가능 |
| 구조화된 프롬프트 | 문제정의-목표-비목표-성공기준-실행계획 구조로 변환됨 |
| 작업 품질 향상 | FCSR 10%+ 향상 (첫 시도 성공률) |

---

## 완료 조건

- [ ] `prompt-enhancer` skill 호출 방법 숙지
- [ ] 간단한 요청을 구조화된 프롬프트로 변환 경험
- [ ] Skill 활용 전/후 결과 비교 이해

---

## 핵심 가치

> **"간단한 프롬프트 + Skill = 구조화된 완벽한 요청"**

막연한 요청을 AI가 정확히 이해할 수 있는 형태로 변환하여
**첫 시도에 성공**할 확률을 극대화한다.
