# Outcome: FCSR 측정 및 추적

## 🎯 목표

**The One Number: FCSR 80% 달성 및 유지**

## 📊 측정 기준

| 지표 | 목표 |
|------|------|
| FCSR (First-Command Success Rate) | 80% 이상 |
| 주간 측정 빈도 | 매주 금요일 |
| 개선 추세 | 월 +5%p |

## 💡 왜 중요한가?

```
FCSR = 첫 번째 명령 성공률

FCSR 20%:
10개 작업 → 8개 재작업 → 하루 종일 삽질
⏱️  8시간 근무 → 실제 산출물 2시간

FCSR 80%:
10개 작업 → 2개만 재작업 → 빠른 진행
⏱️  8시간 근무 → 실제 산출물 6시간
```

## ✅ 성공 상태

- [ ] FCSR 계산법 이해
- [ ] 매일 작업 기록
- [ ] 주간 FCSR 80% 달성
- [ ] 개선 트렌드 파악
- [ ] 병목 구간 식별

## 🎬 기대 효과

- **생산성 3배:** 재작업 시간 80% 감소
- **예측 가능:** 작업 시간 정확히 추정
- **지속 개선:** 데이터 기반 개선
- **팀 벤치마크:** 전체 팀 FCSR 추적

## 📐 계산 공식

```
FCSR = (첫 시도 성공 작업) ÷ (전체 작업) × 100%

예시:
오늘 10개 작업, 8개 한 번에 성공
→ FCSR = 8 ÷ 10 × 100% = 80% ✅
```
