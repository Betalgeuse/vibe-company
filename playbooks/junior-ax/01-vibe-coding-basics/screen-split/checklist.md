# Checklist: 화면 분할 환경 설정

> ⚠️ **환경 세팅 전 확인사항** - 한 번 설정하면 평생 사용!

## ✅ 도구 선택

### 1. 본인 레벨 확인
- [ ] 초보자 → VS Code 선택
- [ ] 중급자 → iTerm2 + Tmux 선택
- [ ] 고급자 → Neovim + Tmux 선택

### 2. 도구 설치 확인
```bash
# VS Code
code --version

# iTerm2 (Mac만)
ls /Applications/iTerm.app

# Tmux
tmux -V
```
- [ ] 선택한 도구가 설치되어 있나?
- [ ] 최신 버전인가?

---

## ✅ 레이아웃 설정

### 3. 표준 레이아웃 결정
```
추천 레이아웃:
├─ 왼쪽: 코드 (60%)
├─ 우상: 실행 결과 (40%)
└─ 하단: 터미널 (20%)
```
- [ ] 3분할 구조 결정했나?
- [ ] 각 영역 크기 비율 정했나?

### 4. 키보드 단축키 설정
**VS Code:**
- [ ] `Cmd+1, 2, 3` 에디터 이동 테스트
- [ ] `Ctrl+\`` 터미널 토글 테스트
- [ ] `Cmd+B` 사이드바 토글 테스트

**Tmux:**
- [ ] `Ctrl+b %` 수직 분할 테스트
- [ ] `Ctrl+b "` 수평 분할 테스트
- [ ] `Ctrl+b ↑↓←→` 영역 이동 테스트

---

## ✅ 실전 테스트

### 5. 실제 작업으로 테스트
```bash
# 영역 1: 코드 열기
code . (또는 vim .)

# 영역 2: 개발 서버 실행
npm run dev

# 영역 3: Git 상태 확인
git status
```
- [ ] 3개 영역 모두 보이나?
- [ ] 키보드만으로 이동 가능한가?
- [ ] 마우스 없이 작업 가능한가?

### 6. Alt+Tab 금지 테스트
**1시간 작업하면서:**
- [ ] Alt+Tab을 누르지 않았나?
- [ ] 마우스로 창 클릭하지 않았나?
- [ ] 모든 정보가 한 화면에 보이나?

---

## ✅ 자동화 설정

### 7. 시작 스크립트 생성 (선택)
```bash
# ~/scripts/vibe-setup.sh 생성했나?
chmod +x ~/scripts/vibe-setup.sh
./vibe-setup.sh
```
- [ ] 스크립트가 정상 작동하나?
- [ ] 한 번에 환경이 세팅되나?
- [ ] 프로젝트별 스크립트 만들었나?

## 🚨 트러블슈팅

### ❌ "화면이 너무 작아요"
**해결:**
```
- 13인치: 2분할만 사용 (코드 + 터미널)
- 15인치+: 3분할 가능
- 외장 모니터: 4분할 추천
```

### ❌ "단축키가 안 먹혀요"
**해결:**
```
1. 다른 앱 단축키와 충돌 확인
2. 시스템 환경설정 → 키보드 → 단축키
3. 충돌하는 단축키 비활성화
```

### ❌ "Tmux pane 크기 조절이 안돼요"
**해결:**
```bash
# Pane 크기 조절
Ctrl+b :resize-pane -D 5  # 아래로 5줄
Ctrl+b :resize-pane -U 5  # 위로 5줄
Ctrl+b :resize-pane -L 5  # 왼쪽 5칸
Ctrl+b :resize-pane -R 5  # 오른쪽 5칸
```
