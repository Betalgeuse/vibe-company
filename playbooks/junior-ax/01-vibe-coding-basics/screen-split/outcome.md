# Outcome: 화면 분할 환경 설정

## 🎯 목표

**창 전환 0회 달성 - 모든 정보를 한 화면에**

## 📊 측정 기준

| 지표 | Before | After |
|------|--------|-------|
| 창 전환 횟수/시간 | 50-100회 | 0회 |
| Alt+Tab 사용 | 매번 | 없음 |
| 마우스 사용 | 많음 | 최소화 |
| 집중력 유지 | ❌ | ✅ |

## 💡 왜 중요한가?

```
Before (창 전환):
코드 작성 → Alt+Tab → 터미널 확인 → Alt+Tab → 브라우저 확인 → Alt+Tab
⏱️  매번 2-3초 소요 × 100회/일 = 5-10분 낭비
😵 집중력 매번 깨짐

After (화면 분할):
모든 것을 한 화면에 → 눈만 움직임 → 키보드 단축키로 이동
⏱️  0초
😎 집중력 유지
```

## ✅ 성공 상태

- [ ] 화면이 2-3개 영역으로 분할됨
- [ ] 코드, 터미널, 브라우저가 동시에 보임
- [ ] 키보드 단축키로 영역 간 이동 가능
- [ ] 마우스 없이 작업 가능
- [ ] Alt+Tab을 하루 종일 안 누름

## 🎬 기대 효과

- **시간 절약:** 하루 10분 → 주 50분 → 월 200분 (3시간+)
- **몰입 유지:** Context Switching 최소화
- **생산성:** 창 찾는 시간 0초
- **피로 감소:** 마우스 사용 90% 감소
