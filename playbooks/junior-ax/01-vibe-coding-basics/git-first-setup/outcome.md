# Outcome: Git 첫 설정

## 🎯 목표

**Git 계정 생성부터 저장소 클론까지 완벽 마스터**

## 📊 측정 기준

| 단계 | 상태 | 소요 시간 |
|------|------|----------|
| Git 계정 생성 | ✅ | 5분 |
| Git 설치 | ✅ | 10분 |
| 저장소 클론 | ✅ | 2분 |
| 경로 설정 완료 | ✅ | 1분 |

## 💡 왜 중요한가?

```
Before:
😵 Git이 뭐지? → 😰 어떻게 시작하지? → 🤯 포기

After:
😎 계정 생성 → ✅ 저장소 클론 → 🚀 바로 코딩 시작
```

## ✅ 성공 상태

- [ ] GitHub 계정이 있음
- [ ] Git이 내 컴퓨터에 설치됨
- [ ] 원하는 저장소를 내 컴퓨터로 복사했음
- [ ] 저장소 위치를 정확히 알고 있음
- [ ] `git status` 명령이 작동함

## 🎬 기대 효과

- **시작 장벽 제거:** 복잡한 설정 없이 빠르게 시작
- **협업 준비 완료:** 팀의 코드를 받을 수 있음
- **Vibe Coding 시작:** 이제 AI와 코딩할 준비 완료
