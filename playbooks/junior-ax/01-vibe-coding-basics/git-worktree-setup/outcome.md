# Outcome: Git Worktree 설정

## 🎯 목표

**브랜치 전환 시간 0초 달성**

## 📊 측정 기준

| 지표 | Before | After |
|------|--------|-------|
| 브랜치 전환 시간 | 10-30초 | 0초 |
| 창 전환 횟수 | 매번 | 0회 |
| 집중력 유지 | ❌ | ✅ |

## 💡 왜 중요한가?

```
Before (git checkout):
코딩 중 → 브랜치 전환 → ⏳ 로딩 → 😵 집중력 깨짐

After (git worktree):
코딩 중 → 다른 폴더로 이동 → 즉시 작업 → 😎 집중력 유지
```

## ✅ 성공 상태

- [ ] 여러 브랜치를 동시에 열어둘 수 있음
- [ ] 브랜치 전환 시 로딩 시간 없음
- [ ] 각 브랜치가 독립적인 폴더로 존재
- [ ] 한 화면에서 여러 브랜치 동시 작업 가능

## 🎬 기대 효과

- **생산성 향상:** 브랜치 전환 대기 시간 제거
- **몰입 유지:** Context Switching 최소화
- **병렬 작업:** 여러 브랜치 동시 개발 가능
