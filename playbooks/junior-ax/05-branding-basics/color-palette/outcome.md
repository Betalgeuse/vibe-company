# Outcome: Color Palette (컬러 팔레트)

## 🎯 목표

**브랜드 색상 시스템을 30분 안에 정의하고 Tailwind CSS에 적용**

## 📊 측정 기준

| 지표 | Target |
|------|--------|
| 컬러 팔레트 생성 | Primary, Secondary, Accent |
| Tailwind config 설정 | 완료 |
| 접근성 검증 | WCAG AA 이상 |
| 전체 소요 시간 | 30-45분 |

## 💡 왜 중요한가?

```
색상 없이:
기본 파란색, 회색 → 개성 없음

색상 시스템 있으면:
일관된 컬러 → 브랜드 인지 → 전문성
```

## ✅ 성공 상태

- [ ] Primary 색상 선정 (+ 음영 9단계)
- [ ] Secondary 색상 선정 (선택)
- [ ] Accent 색상 선정
- [ ] Gray scale 정의
- [ ] Tailwind config 업데이트
- [ ] 접근성 검증 완료 (대비 4.5:1 이상)

## 🎬 기대 효과

- **일관성:** 모든 페이지에서 같은 색상
- **접근성:** 시각 장애인도 사용 가능
- **효율:** Tailwind로 즉시 적용
- **확장성:** 다크 모드 자동 지원

## 🎨 컬러 시스템 구성

### 1. Primary (주조색)
```
브랜드 대표 색상
용도: 버튼, 링크, 강조
```

### 2. Secondary (보조색)
```
Primary 보완 색상
용도: 서브 요소, 배경
```

### 3. Accent (강조색)
```
특별한 액션 강조
용도: CTA, 알림, 배지
```

### 4. Neutral (중립색)
```
Gray scale 9단계
용도: 텍스트, 배경, 테두리
```

## 📋 색상 선택 기준

### Must-have (필수)
- [ ] 브랜드 컨셉과 일치
- [ ] 접근성 기준 충족 (WCAG AA)
- [ ] 라이트/다크 모드 모두 작동
- [ ] 경쟁사와 차별화

### Nice-to-have (선택)
- [ ] 색상 심리학 고려
- [ ] 문화적 의미 고려
- [ ] 트렌드 반영

## 🚨 피해야 할 것

❌ **색상 너무 많음:** 5개 이상  
❌ **접근성 무시:** 대비 낮음  
❌ **경쟁사 그대로 복사**  
❌ **트렌드만 따라가기**
