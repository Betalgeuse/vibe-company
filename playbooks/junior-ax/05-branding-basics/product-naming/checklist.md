## 🚨 트러블슈팅

### ❌ 도메인 모두 불가능
**증상:** 10개 후보 모두 도메인 없음

**해결:**
```
1. ChatGPT에게 재요청:
   "도메인 확보 가능성이 높은 이름으로 10개 더"

2. 도메인 Hack 시도:
   - 국가 도메인 활용 (예: bit.ly)
   - 조합 도메인 (예: get[name].com)

3. 차선책:
   - .io, .ai 우선 고려
   - Premium 도메인 구매 검토
```

### ❌ 팀원 의견 불일치
**증상:** 상위 3개 후보 의견 엇갈림

**해결:**
```
1. 투표:
   - 각자 1-3위 선정
   - 점수 합산

2. 엄마 테스트:
   - 전화로 설명 가능한가?
   - 가장 쉬운 것 선택

3. 타이 브레이커:
   - 도메인 가격 낮은 것
   - 로고 만들기 쉬운 것
```

### ❌ 30분 초과
**증상:** 시간 내 결정 못함

**해결:**
```
❌ 더 고민하기
✅ 강제 결정!

이유:
- 완벽한 이름은 없음
- 나중에 바꿀 수도 있음
- 빠른 실행이 더 중요

→ 상위 3개 중 동전 던지기!
```

### ❌ 의미 없는 이름들
**증상:** AI가 생성한 이름이 모두 이상함

**해결:**
```
프롬프트 개선:

"다음 성공 사례를 참고하세요:
- Notion: 개념, 모든 것을 담는다
- Slack: 협업, 대화 플랫폼
- Linear: 직선적, 효율적 워크플로우

[내 제품 정보]
위 사례처럼 의미 있는 이름 10개"
```
