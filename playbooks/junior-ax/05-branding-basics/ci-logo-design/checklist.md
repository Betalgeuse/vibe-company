## ✅ Step 1: 프롬프트 작성 (10분)

### 프롬프트 구성 요소
- [ ] 로고 유형 명시
  ```
  예: "lettermark logo" / "minimal symbol logo"
  ```

- [ ] 브랜드명 포함
  ```
  예: "for CodeCraft"
  ```

- [ ] 키워드 3개 추가
  ```
  예: "technical, trustworthy, clean"
  ```

- [ ] 색상 방향
  ```
  예: "blue and purple gradient"
  ```

- [ ] 스타일 키워드
  ```
  예: "minimalist, modern, flat design, vector"
  ```

- [ ] 배경 지정
  ```
  예: "white background, transparent"
  ```

### 최종 프롬프트 확인
```
[로고 유형] logo for [브랜드명], 
[키워드 1], [키워드 2], [키워드 3],
[색상], minimalist, modern, vector, simple,
white background, professional --v 6 --s 250
```

- [ ] 프롬프트 복사 완료

## 🚨 트러블슈팅

### ❌ AI 생성 결과가 마음에 안 듦
**증상:** 4개 모두 이상함

**해결:**
```
1. 프롬프트 수정:
   - "more minimal" 추가
   - 색상 구체화 (#3B82F6)
   - 레퍼런스 추가 "similar to [유명 브랜드]"

2. 다른 스타일 시도:
   - "geometric logo"
   - "abstract logo"
   - "tech logo"

3. 재생성:
   - 같은 프롬프트로 3-4번 재시도
   - 각각 다른 결과 나옴
```

### ❌ 축소 시 못 알아봄
**증상:** 32px에서 뭉개짐

**해결:**
```
Figma에서:
1. 선 굵기 증가 (1px → 2px)
2. 여백 확보 (양옆 20%)
3. 디테일 제거 (불필요한 요소)
4. 단순화 (Shape 개수 줄이기)
```

### ❌ SVG 변환 실패
**증상:** Vectorizer.ai가 이상하게 변환

**해결:**
```
Figma에서 직접 제작:
1. Midjourney 이미지를 배경에
2. Opacity 50%로 설정
3. 위에 Shape Tool로 트레이스
4. 원본 이미지 삭제
5. SVG Export

→ 20분 추가 소요되지만 가장 깔끔!
```

### ❌ 파비콘이 안 보임
**증상:** 브라우저 탭에 아이콘 없음

**해결:**
```
1. 브라우저 캐시 삭제:
   - Ctrl + Shift + Delete (Chrome)
   - 하드 리프레시: Ctrl + F5

2. 파일 경로 확인:
   - /public/favicon-32x32.png 존재?
   - layout.tsx에 경로 맞나?

3. 개발 서버 재시작:
   - npm run dev 중단 후 재시작
```
