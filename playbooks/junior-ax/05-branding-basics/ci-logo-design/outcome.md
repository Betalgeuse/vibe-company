# Outcome: CI/Logo Design (CI/로고 제작)

## 🎯 목표

**Midjourney/DALL-E로 45-60분 안에 프로페셔널한 로고 제작**

## 📊 측정 기준

| 지표 | Target |
|------|--------|
| AI 생성 후보 | 4개 이상 |
| 최종 로고 선정 시간 | 45분 이내 |
| 파일 포맷 | SVG + PNG (512x512) |
| 파비콘 제작 | 16x16, 32x32, 192x192 |
| 전체 소요 시간 | 45-60분 |

## 💡 왜 중요한가?

```
로고 없이:
웹사이트, 앱, 명함 → "아마추어" 느낌

로고 있으면:
첫인상 프로페셔널 → 신뢰도 상승 → 브랜드 인지
```

**로고의 역할:**
- ✅ 브랜드 정체성의 핵심
- ✅ 첫인상 결정 (3초 이내)
- ✅ 모든 터치포인트에 사용
- ✅ 기억과 인지의 앵커

## ✅ 성공 상태

- [ ] AI로 생성한 4개+ 후보
- [ ] 최종 로고 1개 선정
- [ ] SVG 파일 확보 (벡터)
- [ ] PNG 파일 (512x512, 투명 배경)
- [ ] 파비콘 3종 (16x16, 32x32, 192x192)
- [ ] 라이트/다크 모드 버전 (선택)

## 🎬 기대 효과

- **브랜드 통일:** 모든 채널에서 일관된 이미지
- **신뢰도:** 프로페셔널한 첫인상
- **인지도:** 기억하기 쉬운 시각적 앵커
- **적용 속도:** SVG로 즉시 웹/앱 적용

## 🎨 좋은 로고의 조건

### Must-have (필수)
1. **단순함:** 5초 안에 그릴 수 있을 정도
2. **기억하기 쉬움:** 한 번 보면 기억
3. **확장성:** 작은 크기에서도 식별 가능
4. **적시성:** 브랜드 컨셉과 일치

### Nice-to-have (선택)
1. **의미:** 브랜드 스토리 담김
2. **독특함:** 경쟁사와 차별화
3. **다목적:** 다양한 배경에서 작동
4. **시대 초월:** 트렌드에 휩쓸리지 않음

## 🎨 로고 유형

### 1. 워드마크 (Wordmark)
```
예: Google, FedEx, Coca-Cola
장점: 브랜드명 즉시 인지
단점: 파비콘으로 사용 어려움
```

### 2. 레터마크 (Lettermark)
```
예: IBM, HP, NASA
장점: 간결, 파비콘 적합
단점: 약자 의미 설명 필요
```

### 3. 심볼/아이콘 (Symbol)
```
예: Apple, Twitter, Nike
장점: 언어 무관, 강력한 인지
단점: 브랜드 구축 시간 필요
```

### 4. 조합마크 (Combination)
```
예: Adidas, Burger King, Lacoste
장점: 유연성 높음
단점: 복잡할 수 있음
```

### 추천: 스타트업 MVP
```
레터마크 + 심볼 조합
→ 파비콘: 심볼만
→ 로고: 심볼 + 워드
```
