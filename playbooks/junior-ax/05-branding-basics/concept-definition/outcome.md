# Outcome: Concept Definition (컨셉 정의)

## 🎯 목표

**3-5개 키워드로 브랜드 정체성을 30분 안에 명확화**

## 📊 측정 기준

| 지표 | Target |
|------|--------|
| 레퍼런스 수집 | 3개 이상 |
| 키워드 추출 | 3-5개 |
| 컨셉 문서 작성 | 1페이지 |
| 전체 소요 시간 | 30-45분 |

## 💡 왜 중요한가?

```
컨셉 없이:
"뭔가 멋있게 만들어주세요" → 방향 없음 → 일관성 없는 디자인

컨셉 기반:
"미니멀, 전문적, 신뢰감" → 명확한 방향 → 일관된 브랜딩
```

**브랜드 컨셉의 역할:**
- ✅ 모든 디자인 결정의 기준점
- ✅ 팀 간 커뮤니케이션 정렬
- ✅ AI 도구에게 명확한 지시
- ✅ 일관된 브랜드 경험

## ✅ 성공 상태

- [ ] 경쟁사/유사 서비스 3개 분석 완료
- [ ] 차별화 포인트 3개 도출
- [ ] 브랜드 키워드 3-5개 선정
- [ ] 컨셉 문서 1페이지 작성
- [ ] 팀원 누구나 이해 가능

## 🎬 기대 효과

- **방향성 확립:** 모든 디자인 의사결정 기준
- **시간 절약:** "이건 우리 브랜드에 안 맞아" 즉시 판단
- **일관성:** 로고, 색상, 폰트 모두 같은 방향
- **AI 활용:** ChatGPT/Midjourney에게 명확한 지시

## 🎨 활용 사례

### 실제 성공 사례

**Airbnb**
- 키워드: Belonging, Community, Trust
- 결과: 따뜻한 색상, 친근한 폰트, 사람 중심 이미지

**Stripe**
- 키워드: Simple, Professional, Developer-first
- 결과: 미니멀 디자인, 기술적인 폰트, 블루-퍼플 그라데이션

**Notion**
- 키워드: Flexible, Calm, Organized
- 결과: 부드러운 색상, 깔끔한 레이아웃, 아이콘 중심

**Linear**
- 키워드: Fast, Precise, Beautiful
- 결과: 다크 모드, 퍼플 액센트, 애니메이션

## 📋 컨셉 구성 요소

### 1. 브랜드 퍼스널리티 (성격)
```
만약 브랜드가 사람이라면?
- 나이: 20대? 30대? 중년?
- 성격: 진지? 장난기? 신중?
- 말투: 격식? 캐주얼? 전문적?
```

### 2. 감성 키워드 (느낌)
```
사용자가 느껴야 할 감정:
- 안심, 신뢰, 편안함
- 흥분, 기대, 호기심
- 전문성, 신뢰, 권위
```

### 3. 비주얼 방향 (보임)
```
디자인 스타일:
- 미니멀 vs 풍성함
- 따뜻함 vs 차가움
- 부드러움 vs 날카로움
```

## 🚨 주의사항

### 피해야 할 것
❌ **너무 추상적:** "혁신적이고 창의적인"  
✅ **구체적:** "기술적이고 신뢰할 수 있는"

❌ **너무 많은 키워드:** 10개  
✅ **집중된 키워드:** 3-5개

❌ **경쟁사 베끼기**  
✅ **경쟁사 분석 후 차별화**

### 일관성 체크
```
모든 키워드가 서로 충돌하지 않는가?

❌ 충돌: "캐주얼하고 전문적인"
✅ 조화: "친근하고 신뢰할 수 있는"
```
