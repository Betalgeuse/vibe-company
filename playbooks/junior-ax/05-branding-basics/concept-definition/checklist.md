- [ ] **간접 경쟁사** 1개 수집
  - [ ] 웹사이트 URL 기록

  - [ ] 주요 색상 기록
  - [ ] 폰트 느낌 기록

- [ ] **벤치마크** 1개 수집
  - [ ] 웹사이트 URL 기록

  - [ ] 주요 색상 기록

## ✅ Step 3: 키워드 도출 (10분)

### 브레인스토밍
**기능적 키워드 (무엇을 하는가):**
- [ ] 최소 5개 단어 나열
- [ ] 핵심 1-2개 선택

**감성적 키워드 (어떤 느낌인가):**
- [ ] 최소 5개 단어 나열
- [ ] 핵심 1-2개 선택

**시각적 키워드 (어떻게 보이는가):**
- [ ] 최소 5개 단어 나열
- [ ] 핵심 1-2개 선택

### 최종 선정
- [ ] 전체 키워드 중 3-5개 선택
- [ ] 키워드 간 충돌 없는지 확인
- [ ] 각 키워드 1문장 설명 작성

**최종 키워드:**
1. [ ] [키워드 1] - [설명]
2. [ ] [키워드 2] - [설명]
3. [ ] [키워드 3] - [설명]
4. [ ] [키워드 4] - [설명] (선택)
5. [ ] [키워드 5] - [설명] (선택)

**시간 체크:** 30분 경과 ⏱️

## ✅ Step 5: 검증 및 저장 (5분)

### 자체 검증
- [ ] 키워드가 서로 조화로운가?
  - 예: "캐주얼 + 전문적" → 충돌?
- [ ] 실행 가능한 컨셉인가?
  - 예: "럭셔리" → 하지만 무료 서비스?
- [ ] 팀원이 이해 가능한가?
  - 다른 사람이 읽고 이해 가능?

### 팀 공유 (선택)
- [ ] 컨셉 문서 팀에게 공유
- [ ] 피드백 받기 (5분 제한)
- [ ] 즉시 반영 가능한 것만 수정

### 문서 저장
- [ ] docs/branding/ 폴더 확인
- [ ] concept.md 파일 생성
- [ ] 내용 복사 붙여넣기
- [ ] Git add + commit

## 🚨 트러블슈팅

### ❌ 레퍼런스 찾기 어려움
**증상:** 비슷한 서비스가 없음

**해결:**
```
1. 범위 넓히기:
   - 같은 타겟 사용자를 가진 다른 제품
   - 같은 분위기를 가진 다른 업종

2. Dribbble 검색:
   - "developer tool"
   - "B2B SaaS"
   - "dashboard design"

3. ChatGPT에게 물어보기:
   "[우리 제품] 같은 느낌의 브랜드 10개 추천해줘"
```

### ❌ 키워드가 너무 비슷함
**증상:** "혁신적, 창의적, 독특한" 등 추상적

**해결:**
```
더 구체적으로:
- "혁신적" → "기술적"
- "창의적" → "실험적"
- "독특한" → "미니멀하지만 컬러풀"

5W1H 질문:
- 어떻게 혁신적인가?
- 왜 창의적이라고 느끼는가?
```

### ❌ 팀원 의견 불일치
**증상:** 키워드에 대한 의견 엇갈림

**해결:**
```
1. 투표:
   - 각 키워드에 투표
   - 상위 3-5개 선택

2. 우선순위 설정:
   - 제품 핵심 가치와 가장 가까운 것
   - 경쟁사와 가장 차별화되는 것

3. A/B 컨셉:
   - 2가지 방향 모두 시도
   - 다음 단계(로고)에서 결정
```

### ❌ 컨셉이 너무 추상적
**증상:** "뭔가 멋있는 느낌" 수준

**해결:**
```
구체화 질문:
1. 만약 브랜드가 사람이라면?
   → 나이, 성격, 말투

2. 만약 브랜드가 색이라면?
   → 따뜻한 색? 차가운 색?

3. 만약 브랜드가 음악이라면?
   → 장르, 템포, 분위기

→ 답변을 키워드로 변환!
```

## 💡 Quick Reference

### 키워드 조합 체크
```
✅ 좋은 조합:
- "전문적 + 친근한" = Stripe
- "빠른 + 아름다운" = Linear
- "유연한 + 조직적인" = Notion

❌ 나쁜 조합:
- "캐주얼 + 권위적"
- "화려한 + 미니멀"
- "장난스러운 + 신뢰감"
```

### 빠른 결정 가이드
```
키워드 선택 우선순위:

1순위: 제품 핵심 가치 표현
2순위: 경쟁사와 차별화
3순위: 시각적 표현 가능
4순위: 팀 동의
5순위: 트렌드

→ 1-2순위가 가장 중요!
```

### 컨셉 → 디자인 연결
```
키워드          → 색상
"신뢰감"        → 블루 계열
"성장"          → 그린 계열
"열정"          → 레드-오렌지 계열
"창의성"        → 퍼플 계열

키워드          → 폰트
"전문적"        → 산세리프
"친근한"        → 라운드 산세리프
"권위적"        → 세리프
"기술적"        → 모노스페이스
```
