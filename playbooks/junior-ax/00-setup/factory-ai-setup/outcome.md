# Outcome: Factory AI 설치

## 🎯 이 작업의 목표

**Factory AI가 설치되어 Custom Droid와 Command를 사용할 수 있는 상태**

## ✅ 완료 기준

- [ ] Factory CLI가 설치됨
- [ ] `factory` 명령어가 터미널에서 실행됨
- [ ] Factory 계정 인증 완료
- [ ] `.factory/droids/` 폴더의 Droid 사용 가능

## 📦 결과물

```bash
# 터미널에서 실행 가능
factory

# Droid 목록 확인
ls .factory/droids/

# Droid 호출 예시
factory @niche-market-hunter "SaaS 아이디어 찾기"
```

## ⏱️ 예상 소요 시간

**10-15분**
