# Outcome: Visual Assets (비주얼 에셋)

## 🎯 목표

**소셜 미디어용 비주얼 콘텐츠를 60분 안에 제작 완료**

## 📊 측정 기준

| 지표 | Target |
|------|--------|
| OG Image | 1개 (1200x630px) |
| Product Screenshots | 3-5장 |
| Demo Video | 30-60초 |
| Social Post Images | 2-3개 (1080x1080px) |
| 제작 시간 | 60분 이내 |

## 💡 왜 중요한가?

```
텍스트만 vs 비주얼 포함:
- Engagement: 3배 증가
- Click-through: 5배 증가
- 기억률: 10배 증가
```

## 🎨 필수 에셋 4가지

### 1. OG Image (1200x630px)
**목적:** 링크 공유 시 미리보기  
**플랫폼:** Twitter, LinkedIn, Slack, Discord

**포함 요소:**
- 제품 로고
- 제품명
- Tagline (한 줄 설명)
- 핵심 스크린샷 또는 일러스트
- 브랜드 색상

**도구:**
- OG Image Playground (https://og-playground.vercel.app/)
- Canva (무료)
- Bannerbear (무료 티어)

---

### 2. Product Screenshots (3-5장)
**목적:** 기능 설명, ProductHunt Gallery  
**플랫폼:** ProductHunt, Twitter, Website

**촬영 대상:**
1. 메인 화면 (Hero)
2. 핵심 기능 #1 (가장 중요)
3. 핵심 기능 #2
4. Dashboard/결과 화면
5. Before/After (선택)

**사양:**
- 해상도: 1920x1080 이상
- 형식: PNG (투명 배경 선택 가능)
- 민감 정보 제거 (테스트 데이터 사용)

**도구:**
- 브라우저 개발자 도구 (반응형 미리보기)
- Cleanshot X (Mac)
- ShareX (Windows)
- Snagit

---

### 3. Demo Video (30-60초)
**목적:** 제품 사용 흐름 시연  
**플랫폼:** ProductHunt, Twitter, Website

**구성:**
1. 문제 제시 (5초)
2. 솔루션 소개 (5초)
3. 핵심 기능 시연 (20-30초)
4. 결과/CTA (5-10초)

**사양:**
- 길이: 30-60초 (최대 90초)
- 해상도: 1920x1080
- 형식: MP4
- 자막: 필수 (음소거 시청 대비)

**도구:**
- Loom (무료, 5분 제한)
- OBS Studio (무료, 무제한)
- ScreenFlow (Mac)
- Camtasia

---

### 4. Social Media Post Images (1080x1080px)
**목적:** Twitter/LinkedIn 포스트 첨부  
**플랫폼:** Twitter, LinkedIn

**유형:**
- Feature highlight
- 사용자 후기 (텍스트 오버레이)
- 통계/데이터 시각화
- Behind-the-scenes

**도구:**
- Canva (무료)
- Figma (무료)
- Adobe Express (무료)

---

- **CTR 증가:** 링크 클릭률 5배

- **ProductHunt:** Gallery 완성도로 Upvotes 증가
- **전문성:** 완성도 높은 브랜드 이미지
