# 토큰 로열티 MVP - Founder Sales Playbook

## 🎯 목표 (Month 1-6)

| Phase | Goal | Activities | Target |
|-------|------|-----------|--------|
| **Month 1-2** | MVP 구축 + 첫 고객 | 개발 + 콜드 아웃바운드 | 3 beta (LOU signed) |
| **Month 3** | Product-Market Fit | Feedback 반복 + 10명 sale | 10 paying, $5K MRR |
| **Month 4-6** | Scale | Sales system 정비 | 15명, $11.6K MRR |

## 📍 ICP Targeting Strategy

### 누구를 찾는가?

**Primary**: 독립 카페 점장/지배인
- 월 매출: $50K-500K (평균 $100-200K)
- 고객층: 직장인 + 대학생 (재방문 가능층)
- 기존 로열티: 종이 쿠폰 or 없음
- Tech 수준: 중간 (POS 사용 중)

**Signal** (찾기 쉬운 신호):
- ✅ 최근 새 매장 오픈 (확장 마인드)
- ✅ IG 팔로워 500+ (고객 기반)
- ✅ "멤버십", "스탬프", "포인트" 언급 (pain point)
- ✅ "대형 체인과 차별화" 관심 표현

**Exclude** (지양):
- ❌ 프랜차이즈 (다른 pitch)
- ❌ 월 매출 <$30K (예산 없음)
- ❌ 완전 디지털 (이미 유사 시스템)

### 어디서 찾는가?

| Channel | Scale | ROI | Notes |
|---------|-------|-----|-------|
| **LinkedIn** | 50+/주 | High | "카페 오너" 검색, 콜드 DM |
| **Google Maps** | 100+/주 | Medium | 강남/홍대/을지로 크롤링 → IG/이메일 |
| **Network** | 10-20/주 | Very High | 친구/지인 소개 (최고 전환율) |
| **Cafe Events** | 30+/이벤트 | High | 카페 협회, 상권 세미나 |
| **Content** | 20+/월 | Low (초기) | SEO/블로그 (Month 4+) |

## ☎️ Discovery Call Framework (15분)

### 구조화된 질문 (반드시 이 순서)

**[Opening - 30초]**
```
"감사합니다 시간 내주셔서.
저는 [당신이름]이고, 
카페 점장님들을 위한 로열티 플랫폼을 만들고 있어요.

오늘은 현재 고객 관리 방식을 이해하고,
우리가 도움이 될지 함께 확인하는 시간이고,
판매 목적은 아닙니다."
```

**[Current State - 2분]**
```
질문 1: "지금 고객 로열티/재방문을 어떻게 추적하세요?"
(답변 들음 - 스탬프/앱/없음 등)

질문 2: "그 방식에서 가장 불편한 점이 뭔가요?"
(답변 들음 - 수동, 비용, 복잡함 등)

질문 3: "그걸 해결하려고 시도한 적 있으세요?"
(답변 들음 - 했으면 왜 안 됐는지, 없으면 왜 미뤘는지)
```

**[Pain Quantification - 2분]**
```
질문 4: "지금 고객 관리에 주당 몇 시간 써요?"
(답변: 보통 3-5시간)

질문 5: "그 시간이 사업에 도움이 되나요?"
(답변: 거의 아님 - "그냥 해야 하니까")

정리: "그럼 주 3시간 × $X/시간 = 월 $XXX 비용이네요.
만약 그걸 자동화하면?"
```

**[Buying Process - 2분]**
```
질문 6: "이런 솔루션을 도입하려면 누가 결정하나요?"
(답변: 보통 자기 자신 or 부점장)

질문 7: "예산은 어디서 나와요?"
(답변: 마케팅 예산 or 사업 계정)

질문 8: "지금 월 로열티/마케팅에 얼마 써요?"
(답변: $0-500)
```

**[Solution Fit - 5분]**
```
"저희 제품을 보여드릴게요.

[Admin Dashboard 보여주기]
• 고객 리스트 + 재방문율
• 토큰 통계

[Mobile App 보여주기]
• 고객이 보는 토큰 잔액
• QR로 상환 (안 돼)

우리가 특별한 건:
1. Price: $499/월 (기존 $2K+)
2. Setup: 30분이면 POS 연동 끝
3. Blockchain: 투명하고 거래 가능
"

반응 관찰:
- 긍정적 → "데모 보여줄까요?"
- 의심적 → "어떤 게 concern인가요?"
```

**[Timeline & Next Steps - 2분]**
```
질문 9: "언제까지 의사결정 가능해요?"
(답변: 이번주, 이번달, 분기말 등)

질문 10: "데모 후 다음 스텝은?"
(답변: 팀과 상의, 14일 체험, 계약 등)

클로징:
"좋아요. 그럼 이렇게 해요:
1. 이번주 금요일 30분 데모 예약 (구체적 시간)
2. 14일 무료 체험 드릴게요
3. 궁금하면 언제든 연락주세요"

(달력 공유 - Calendly 또는 카톡)
```

## 🎬 Demo Script (20분)

1. 고객 실명 + 카페명 정확히 알기
2. 그들의 "pain point" 메모해두기
3. 라이브 데모 (녹화 아님)
4. 테스트 데이터로 실제 시나리오 시뮬레이션

**[Opening - 1분]**
```

기억나세요?

우리는 정확히 그 문제를 이렇게 풉니다:"
```

**[Problem Recap - 2분]**
```
Admin Dashboard 첫 화면 보여주기:

"일반적으로:
• 종이 쿠폰: 분실 위험, 추적 불가
• 기존 로열티: 너무 비쌈
• 수동 관리: 시간 낭비

우리는 이 3가지를 모두 해결합니다."
```

## 🎯 Common Objections & Rebuttals

### 1️⃣ "암호화폐는 고객이 안 쓸 건데..."

**우리 반박**:
```
"아 맞아요, 그래서 우리가 중요한 부분입니다.

고객은 '암호'를 몰라도 됩니다.

고객 입장:
• 결제하면 → 토큰 자동 받음
• 나중에 → QR 스캔해서 사용
• 그게 다.

우리가 '블록체인'을 쓰는 이유는:
• 투명성 (고객이 신뢰)
• 거래 가능 (재미)
• 불변성 (안전)

이 기술을 고객한테는 숨기고,
점장님 입장에서만 "투명하다"로 강조합니다."
```

### 2️⃣ "이미 [경쟁사] 쓰고 있는데..."

**우리 반박**:
```
"어떤 솔루션 쓰고 있으세요?

[고객이 답변]

아, 그거 쓰고 계신 비용이 얼마예요?

[고객이 답변: 보통 $500-2K]

우리가 제안하는 건:
• 같은 기능을 $499/월 (또는 더 저렴)
• + 블록체인 투명성 추가
• + POS 자동 연동

기존 시스템은 계속 쓰고,
우리를 'loyalty 전담'으로 추가하는 거니까
실제로 기능이 더 강합니다."
```

### 3️⃣ "설정이 복잡하지 않나?"

**우리 반박**:
```
"아니요, 오히려 간단합니다.

[실제 과정 보여주기]
1. 웹사이트 들어가서 가입
2. Square/Clover API 키 입력 (1분)
3. "연동 완료!" (자동)
4. 과거 거래 import (자동)
5. 끝. 내일부터 자동 발급.

대부분 30분 안에 끝나고,
질문 있으면 저희가 줌으로 help드립니다.

직원분들 교육도?
→ 바리스타는 뭘 할 필요 없어요.
계산원만 "QR 스캔" 하면 되니까
5분이면 이해합니다."
```

### 4️⃣ "처음 써본 시스템이라 불안해..."

**우리 반박**:
```
"그래서 우리가 14일 무료 제공합니다.

완전히 무료 체험 후:
• "이거 정말 작동해?" 확인
• 직원들도 써볼 수 있고
• 고객들 반응도 볼 수 있고

리스크 정말 없습니다.

대신 한 가지 부탁:
14일 체험 중에 주 1-2회 우리 피드백 반영해주세요.
그럼 더 좋은 경험 드릴 수 있으니까."
```

### 5️⃣ "경제 안 좋은데 지출을 늘릴 수 없어..."

**우리 반박**:
```
"아, 딱 그래서 우리 플랫폼이 필요한 거예요.

지금:
• 수동으로 month 3시간 = $300-500 시간 낭비
• 고객 재방문율 추적 불가 = 마케팅 효율 떨어짐

우리 써서:
• 시간 절약 = $300-500 회수
• 재방문 추적 = 마케팅 ROI 개선
• Month 3에 ROI positive

그래서 "지출"이 아니라 "투자"입니다.
실제로 비용이 줄어드는 거죠."
```
