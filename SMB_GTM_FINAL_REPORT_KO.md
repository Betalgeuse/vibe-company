### 1-3. 고객 가치 평가

#### ✅ 강점
1. **가격 접근성**: 기존 도구 대비 **60-70% 저렴**
2. **온보딩 속도**: No-code 빌더 (3-5분, Shopify 플러그인 형태)

3. **고객 채택 위험**: 초기 고객 "포인트는 포인트"라고 생각

```

```

### 3-5. 블록체인 역차별화

```
왜 블록체인인가?

공급망 추적의 가치:
├─ 데이터 진실성 (위변조 방지)
│  └─ 비용: 블록체인 $$$$ (합의 오버헤드)
│     vs DB $$ (중앙화)
│  └─ 결론: DB가 더 싼데 왜 블록체인? ❌
│
├─ 속도
│  └─ 블록체인: 느림 (10-30초 확정)
│     vs DB: 빠름 (즉시)
│  └─ 신선 식품 추적에는 역효과 ❌
│
└─ 분산성 (다자간 신뢰)
   └─ SMB "로컬 공급망" = 신뢰할 사람 알고 있음 ❌
      (블록체인이 필요 없는 상황)

결론: 블록체인은 **비용 증가, 이점 없음** = 역차별화
```
