## 🔴 3가지 Critical Issues

### 1️⃣ CAC $1,000은 거짓일 가능성 높음

**문제**:
- SMB SaaS 벤치마크: $3-5K (Token은 1/3 수준)
- Founder-led sales 시간 투입량이 현실화되지 않음

**현실 계산**:
```
월 2.5명 고객 필요 (30명/년)
→ 콜드 이메일 1,000-1,500명/월 접근
→ Founder 월 50-70시간 투입

인건비 배분: 월 $5,800
고객 2.5명당 → **CAC $2,320** (실제)
```

**평가**: ❌ **심각한 과소 추정**

---

### 2️⃣ LTV $24K는 3년 기준인데 실제는 1.5-2년

**문제**:
- 카페 + SaaS 결합 churn이 높음

**현실 계산**:
```
SMB SaaS churn: 월 4% (기본값)
카페 폐업율: 월 2% (추가)
결합 churn: 월 5-6%

Lifespan:
- 월 4% churn = 25개월
- 월 6% churn = 16개월

LTV 재계산:
- 기본값 (4%): $16,000 ✅
- 현실값 (6%): $10,000 ⚠️
- 제시값 (3%): $22,500 ❌ 낙관적
```

**평가**: ⚠️ **낙관적 가정. 16K가 기본값**

---

### 3️⃣ 고객 지불 의향이 확실하지 않음

**문제**:
- Square Loyalty: $45-105/월 (기존 POS 고객은 사실상 무료)
- Stamp Me: $49-79/월
- Token Loyalty: $499/월 = **6-10배 비쌈**

**필요한 증명**:
```
월 $500 추가 비용 정당화
→ 월 $500 이상 매출 증가 필요

카페 월 거래액 $5,000 기준
→ 10% 매출 증가 필요
→ 로열티 프로그램으로 가능한가?
```

**평가**: ❌ **증명 부재. 위험**
