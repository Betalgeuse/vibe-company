## 🚨 치명적 약점 7가지

### 1️⃣ TAM 미검증 (CRITICAL)
- **주장**: $800M-1.2B 원료 소싱 시장
- **현실**: 데이터 없음, 실제는 $1.5B-3.7B일 가능성
- **근거**: K-뷰티 완성품 시장 $14.61B × 10-25% = $1.5-3.7B (원재료)
- **영향**: SAM 과대평가 → 시장 규모 절반일 가능성

### 2️⃣ ICP 규모 미검증 (CRITICAL)
- **주장**: 한국 K-뷰티 브랜드 600개 (10-100명)
- **현실**: 웹서치 결과 검증 불가 (구체적 수치 없음)
- **근거**: 거의 모든 문서화된 브랜드는 100명 미만이거나 1000명+ 규모
- **영향**: 실제 타겟 기업은 50-100개일 가능성

### 3️⃣ 영업 사이클 과소평가 (CRITICAL)
- **주장**: 2-3주 클로징 (발견 → 데모 → 체험 → 계약)
- **현실**: B2B 엔터프라이즈 SaaS는 3-6개월
- **근거**: CEO/Finance/IT 의사결정위원회 검토 필요
- **영향**: CAC $4.5K → 실제 $9K-18K (유닛 이코노믹스 파괴)

### 4️⃣ 공급사 데이터 지속성 미해결 (CRITICAL)
- **주장**: "초기 수동 입력 + v1.5 API 자동화"
- **현실**: Alibaba 크롤링 위반(ToS), 수동 입력 40-60시간, 지속 불가능
- **근거**: 공급사 참여 없이는 데이터가 정적(stale)
- **영향**: 플랫폼의 핵심 가치 = 실시간 가격 데이터 → 없으면 제품 실패

### 5️⃣ 고객 이탈 과소평가 (CRITICAL)
- **주장**: 5%/분기 (20%/년)
- **현실**: 일반 B2B SaaS 8-10%/월 (100%+/년) until sticky
- **근거**: 전환 비용 없음 (Alibaba 항상 가능), 점착성 기능 로드맵에만 있음
- **영향**: LTV:CAC 10:1 → 실제 2-3:1

### 6️⃣ GTM 검증 없음 (CRITICAL)
- **주장**: "Founder-led sales playbook 가능"
- **현실**: 고객 인터뷰 0건, 세일즈 경험 제한적
- **근거**: MVP 문서 "Domain Expertise: ⚠️ Partial", "Sales Ability: ⚠️ Partial"
- **영향**: Year 1 50명 고객 계획은 근거 없음

### 7️⃣ 규제/데이터보안 위험 과소평가 (MEDIUM-HIGH)
