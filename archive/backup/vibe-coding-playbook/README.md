**Author:** Buyback Architect

## 📖 소개

Vibe Coding은 AI 시대의 생산성 방법론입니다. 
몰입 상태를 유지하며 한 번에 정확한 결과를 만들어내는 기술을 배우세요.


## 🎯 핵심 지표: FCSR 80%

**First-Command Success Rate (첫 번째 명령 성공률)**  
10번 작업하면 8번은 한 번에 성공하는 것이 목표입니다.
