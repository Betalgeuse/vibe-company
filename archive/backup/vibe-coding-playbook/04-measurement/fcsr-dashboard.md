# 📊 FCSR 측정 대시보드

**데이터로 성장하기**

FCSR(First-Command Success Rate)을 측정하고 개선하는 대시보드입니다.


## 📈 일일 측정 템플릿

```markdown
## FCSR 일일 기록

날짜: 2026-01-17

### 작업 목록
1. 로그인 페이지 구현 → ✅ 첫 시도 성공
2. API 연동 → ❌ 2번 시도 후 성공
3. 에러 처리 → ✅ 첫 시도 성공
4. CSS 수정 → ❌ 3번 시도 후 성공
5. 테스트 작성 → ✅ 첫 시도 성공

### 결과
- 전체 작업: 5개
- 첫 시도 성공: 3개
- **FCSR: 60%**

### 분석
- Reference를 충분히 찾지 않았음
- Prompt가 너무 짧았음

### 다음 개선
- Reference 3개 이상 찾기
- Prompt 4요소 모두 포함하기
```
