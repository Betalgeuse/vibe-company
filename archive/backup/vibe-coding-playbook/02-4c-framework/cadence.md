# ⏰ Cadence (리듬 및 빈도)

**언제, 얼마나 자주?**

언제, 얼마나 자주 해야 하는지에 대한 가이드입니다.


## 💡 왜 Cadence가 중요한가?

**비유**: 운동 루틴처럼, 언제 무엇을 해야 하는지 정해두면 헷갈리지 않습니다.

```
Cadence 없음             Cadence 있음
     ↓                        ↓
"언제 하지?"            "매일 9시!"
     ↓                        ↓
 까먹고 안 함            자동으로 실행
     ↓                        ↓
 일관성 없음             꾸준히 유지
```

## 📅 Cadence 유형


### 1. Always (항상 유지할 것)

**터미널 분할 상태 유지**

작업을 시작할 때마다 화면을 분할해두세요.

#### 세팅 스크립트 예시

```bash
#!/bin/bash
# vibe-setup.sh

# tmux 세션 시작
tmux new-session -d -s vibe

# Pane 1: Editor
tmux send-keys -t vibe "cd ~/project && vim ." C-m

# Pane 2: Server Log
tmux split-window -h -t vibe
tmux send-keys -t vibe "cd ~/project && npm run dev" C-m

# Pane 3: Git Operations
tmux split-window -v -t vibe
tmux send-keys -t vibe "cd ~/project && git status" C-m

# Attach
tmux attach -t vibe
```

#### 사용 방법

```bash
# 작업 시작 시
./vibe-setup.sh
```

## 📊 Cadence 설정 방법


### Step 1: 작업 분류

작업을 빈도별로 분류하세요:

| 빈도 | 예시 | 설정 방법 |
|------|------|----------|
| **매일** | 환경 세팅, FCSR 측정 | 아침 루틴에 포함 |
| **매주** | 코드 리뷰, 리팩토링 | 특정 요일 지정 |
| **매월** | 의존성 업데이트, 보안 점검 | 달력에 알림 설정 |
| **필요시** | 신규 기능 개발, 버그 수정 | 이슈 발생 시 |


### Step 2: 구체적 시간 지정

❌ **나쁜 예**: "가끔", "틈날 때", "나중에"  
✅ **좋은 예**: "매일 오전 9시", "매주 금요일 오후 3시"

```
모호한 표현               명확한 시간
    ↓                        ↓
안 하게 됨               습관으로 정착
```


### Step 3: 자동화 설정

가능한 것은 자동화하세요!

#### Cron Job 예시

```bash
# 매일 오전 9시: 개발 환경 자동 세팅
0 9 * * * /Users/username/vibe-setup.sh

# 매주 월요일 오전 10시: 주간 리포트 생성
0 10 * * 1 /Users/username/weekly-report.sh

# 매월 1일: 의존성 업데이트 알림
0 0 1 * * /Users/username/dependency-check.sh
```

## 💡 Cadence 유지 Tips


### Tip 1: 달력에 등록
- Google Calendar, Outlook 등 사용
- 알림 15분 전 설정
- 반복 일정으로 등록

### Tip 2: 체크리스트 활용
- 매번 체크리스트 사용
- 놓치는 것 방지
- 진행률 시각화

### Tip 3: 팀과 동기화
- 팀원들과 같은 Cadence 공유
- 서로 리마인드
- 함께 실행하면 지속 가능

### Tip 4: 유연하게 조정
- 완벽하게 지키려 하지 말기
- 안 맞으면 시간 조정
- 중요한 건 꾸준함

## 🚨 흔한 실수


### 실수 1: 너무 많은 Cadence 설정
❌ 매일 10개의 루틴  
✅ 매일 2-3개 핵심 루틴

### 실수 2: 모호한 시간 설정
❌ "아침에", "점심 먹고"  
✅ "오전 9시", "오후 1시"

### 실수 3: 자동화 없이 수동으로만
❌ 매번 기억해서 실행  
✅ 알림/자동화로 보조

### 실수 4: 한 번에 완벽하게
❌ 첫날부터 모든 Cadence 설정  
✅ 하나씩 추가하며 습관화
